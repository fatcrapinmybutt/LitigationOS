## PURPOSE

This document outlines how to ENFORCE judgments and orders once obtained. Winning is only half the battle — collection and compliance enforcement are critical, especially against defendants with histories of non-compliance.

## I. CUSTODY ORDER ENFORCEMENT (Lane A)

### A. Parenting Time Enforcement

**If Watson violates restored parenting time:**

1. **Document every violation** — Date, time, what happened, witnesses, communications
2. **File Motion for Contempt** (Packet 02 template) — MCR 3.208(B)
3. **Request makeup parenting time** — MCL 552.642(5): court SHALL order makeup time
4. **Request compensatory damages** — MCR 3.208(B)(3): court may order payment of costs
5. **Request custody modification** — Persistent interference = changed circumstances under *Vodvarka*
6. **FOC enforcement** — File FOC complaint (free) → FOC investigates → recommends to court

**Escalation ladder:**
```
Violation 1 → Document + FOC complaint
Violation 2 → Motion for contempt + makeup time
Violation 3 → Motion for contempt + fine + modify custody
Violation 4+ → Motion for sole custody + supervised exchange
```

### B. Child Support Modification

Once custody is restored, Andrew should file to:
- Modify support based on new custody arrangement (50/50 = each parent pays proportional)
- Seek reimbursement for support paid during 569+ days of wrongful denial
- Request Watson pay Andrew's support if custody shifts to primary

**Authority:** MCL 552.517, MCL 552.605 (modification based on change of circumstances)

### C. Contempt Purge

For existing contempt orders against Andrew:
- File motion to set aside contempt (void orders issued without due process)
- MCR 2.612(C)(1)(d): Relief from judgment — "the judgment is void"
- Argue: contempt was punitive (not coercive) → requires criminal due process protections
- *In re Contempt of Henry*, 282 Mich App 656 (2009)

## II. HOUSING JUDGMENT ENFORCEMENT (Lane B)

### A. Money Judgment Collection

**If court awards damages against Housing defendants:**

1. **Judgment Lien** — Automatically attaches to defendant's real property in county (MCL 600.6017)
2. **Garnishment** — File garnishment against bank accounts/income (MCR 3.101)
   - Writ of garnishment: MC 14 (Garnishment — Periodic)
   - Post-judgment garnishment: MC 58
3. **Supplemental Proceedings** — MCR 2.621: examine debtor's assets under oath
4. **Asset Discovery** — Subpoena bank records, property records, corporate filings
5. **Execution on Property** — MCL 600.6001: writ of execution on personal/real property

### B. Corporate Veil Piercing

Housing defendants include multiple LLCs (Shady Oaks MHP LLC, Shady Enterprises LLC, North Morris Operating Company). If corporate entities cannot satisfy judgment:

- Pierce corporate veil under *Foodland Distributors v Al-Naimi*, 220 Mich App 453 (1996)
- Three-factor test: (1) unity of interest, (2) unjust consequences, (3) fraud/wrong
- Reach personal assets of Kim Shady and Elise Martin
- Corporate disclosure (filed in Packet 08) establishes ownership chain

### C. Injunctive Enforcement

If court orders repairs/compliance:
- Monitor compliance deadlines
- File show-cause motion for non-compliance
- Request court-appointed receiver for property if landlord refuses repairs
- Report to EGLE for environmental violations (separate track)

## III. FEDERAL §1983 JUDGMENT ENFORCEMENT

### A. Monetary Judgment

Federal money judgments are enforceable in any state:
- Register judgment in Michigan under 28 USC § 1963
- Federal garnishment under FRCP 69
- Contempt powers of federal court are broader than state

### B. Injunctive Relief

Federal injunctions are enforced through:
- Contempt of court (18 USC § 401)
- Civil contempt: coercive daily fines until compliance
- Criminal contempt: if willful violation
- Federal marshals enforce court orders

### C. Consent Decree Monitoring

If case settles with consent decree:
- Retained jurisdiction for compliance monitoring (typically 3-5 years)
- Compliance reports required from defendants
- Motion to enforce for any violation
- Attorney fees for enforcement actions (42 USC § 1988)

## IV. JTC / AGC ENFORCEMENT

### A. JTC Complaint Follow-Up

- JTC investigation is confidential until formal complaint issued
- If JTC acts: public hearing → possible removal, suspension, or censure
- If JTC declines: request reconsideration with additional evidence
- Alternative: File with Michigan Judicial Council for administrative review

### B. AGC Complaint Follow-Up

- AGC investigation: 60-180 days
- If AGC acts: formal charges → Attorney Discipline Board hearing
- Possible outcomes: reprimand, suspension, disbarment
- If AGC declines: request review by Review Board
- Track AGC case number and follow up every 90 days

## VI. PRACTICAL ENFORCEMENT TIPS FOR PRO SE

1. **File judgment immediately** with County Clerk — creates automatic lien
2. **Record judgment** in every county where defendant owns property
3. **Use post-judgment discovery** — You have the RIGHT to discover assets
4. **Garnish early** — Don't wait. File garnishment within 30 days of judgment
5. **Watch for asset transfers** — MCL 566.34 (fraudulent transfer): if defendant moves assets to avoid judgment, you can void the transfer
6. **Renew judgment** — Michigan judgments last 10 years (MCL 600.5809(3)), renewable for 10 more
7. **Interest accrues** — Post-judgment interest at statutory rate (MCL 600.6013)
8. **Keep records** — Track every payment, every violation, every contact

## VII. TIMELINE OF ENFORCEMENT ACTIONS

```
Judgment Day + 0:    File judgment with Clerk → automatic lien
Judgment Day + 7:    Send demand letter to defendant (certified mail)
Judgment Day + 21:   If no payment → file garnishment + supplemental proceedings
Judgment Day + 30:   Garnishment hearing → court orders garnishment
Judgment Day + 60:   If still unpaid → writ of execution on property
Judgment Day + 90:   If assets hidden → motion for fraudulent transfer
Judgment Day + 180:  If still unpaid → contempt proceedings
Judgment Day + 365:  Annual review — renew garnishments, update asset search
```
