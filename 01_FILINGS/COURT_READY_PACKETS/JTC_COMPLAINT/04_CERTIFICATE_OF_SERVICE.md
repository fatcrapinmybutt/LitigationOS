## MAILING INSTRUCTIONS

### Filing Method
The Michigan Judicial Tenure Commission accepts complaints **by mail only**.

### Filing Checklist

- [ ] Print complete complaint package (Documents 01–05)
- [ ] Print all referenced exhibits
- [ ] Have the Verification in Document 01 notarized before a Notary Public
- [ ] Make one complete copy of the entire package for personal records
- [ ] Mail the original package via **USPS Certified Mail, Return Receipt Requested**
- [ ] Mail courtesy copy to Judge McNeill via First-Class Mail
- [ ] Retain certified mail receipt and tracking number
- [ ] Note date of mailing on Cover Letter and this Certificate
- [ ] File green return receipt card when received

### Important Notes

1. **No filing fee.** The JTC does not charge a filing fee.
2. **Confidentiality.** JTC proceedings are confidential under MCR 9.261. Do not publish or distribute this complaint publicly.
3. **Notarization required.** The Verification in Document 01 must be signed and notarized before mailing.
4. **Supplemental filings.** If additional evidence becomes available, a supplemental complaint may be filed at the same address.

### Recommended Mailing Format

- **Envelope:** 9×12 or 10×13 manila envelope
- **Postage:** USPS Certified Mail with Return Receipt (green card)
- **Label:** Address to "Executive Director, Michigan Judicial Tenure Commission"
- **Contents:** Original complaint package with notarized verification and all exhibit copies
