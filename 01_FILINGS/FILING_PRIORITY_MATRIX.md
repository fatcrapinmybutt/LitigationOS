## GO/NO-GO Criteria

A filing package is **GO** when ALL of the following exist:

- [x] Lead document (motion, brief, complaint, or petition)
- [x] Proposed order (where required)
- [x] Proof of service / service certificate
- [x] Exhibit package with index
- [x] All placeholders resolved (no `[ANDREW_REQUIRED]` tags)
- [x] Pre-filing QA sweep passed
- [x] Correct court form used (SCAO where required)
