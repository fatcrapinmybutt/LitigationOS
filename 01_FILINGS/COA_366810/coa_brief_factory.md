## Core constraints
- Statement of Facts uses pinned EvidenceAtoms only.
- Each proposition in Argument requires an authority triple with pinpoint.
- FILE_READY requires QuoteLock verification.

## Assembly outline
1. Cover and compliance
2. Jurisdiction statement
3. Questions presented
4. Standard of review table
5. Statement of facts (pinned)
6. Argument blocks per issue
7. Relief requested
8. Appendix plan and record citations map

## Argument block pattern
- Issue ID
- Standard of review (authority pin)
- Governing law (authority pins)
- Record facts (evidence pins)
- Error analysis (findings gap and contradictions)
- Prejudice and remedy
- Preservation statement
