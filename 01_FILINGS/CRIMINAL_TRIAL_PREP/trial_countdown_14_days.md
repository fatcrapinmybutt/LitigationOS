### During Trial — Key Reminders
1. **Opening statement:** Andrew feared for his life based on a corrections officer's
   warning. The State cannot disprove self-defense beyond a reasonable doubt.
2. **Body camera is king:** Get it admitted early. Let the jury hear the officer's words.
3. **Self-defense burden:** Remind jury — prosecution must disprove, not defense must prove.
4. **Stay calm:** Courts respect composure. Judges respect preparation.
5. **Object when needed:** MRE 404(b), MRE 403, hearsay, relevance.
6. **Closing argument:** "The officer himself told Andrew his life was in danger.
   Andrew did what any reasonable person would do."

### Common Objections to Be Ready For
| Objection | Rule | When to Use |
|-----------|------|-------------|
| Relevance | MRE 401/402 | When prosecution introduces unrelated case info |
| Unfair prejudice | MRE 403 | When probative value outweighed by prejudice |
| Prior bad acts | MRE 404(b) | When prosecution references other cases/history |
| Hearsay | MRE 801/802 | When out-of-court statements offered for truth |
| Leading question | MRE 611(c) | When prosecution leads own witnesses on direct |
| Speculation | MRE 602 | When witness testifies beyond personal knowledge |
| Best evidence | MRE 1002 | When prosecution uses excerpt instead of full video |

## FALLBACK PLANS

### If Body Camera Not Received by Day 7
1. File **Emergency Motion to Compel** production
2. File **Motion for Continuance** — cannot prepare defense without key evidence
3. File **Motion to Dismiss** — Brady violation argument
4. Argue: prosecution's failure to produce exculpatory evidence violates due process

### If Trial Proceeds Without Counsel
1. Invoke **Faretta** right formally on record
2. Request **standby counsel** (*McKaskle v. Wiggins*)
3. Request **brief continuance** for pro se preparation (MCR 6.004)
4. Proceed with trial_prep_package.md as guide

### If Motion for Continuance Denied
1. Proceed to trial with available preparation
2. **Make a record** — state on record that counsel did not communicate,
   Brady material not received, insufficient preparation time
3. This preserves issues for appeal
4. Request standby counsel even if denied continuance

### If Convicted
1. File **Motion for New Trial** (MCR 6.431) within 21 days
   - Grounds: ineffective assistance, Brady violations, newly discovered evidence
2. File **Claim of Appeal** within 42 days (MCR 7.204(A))
3. Request SADO (State Appellate Defender Office) appointment
4. Preserve all issues raised in pretrial motions
