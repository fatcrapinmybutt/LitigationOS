## 6. Conversion Pipeline

### MD → PDF Conversion Required

All filing documents are currently in Markdown (.md) format. Before e-filing:

1. **Convert to DOCX** (intermediate): Use pandoc or similar tool
   ```
   pandoc MOTION.md -o MOTION.docx --reference-doc=template.docx
   ```
2. **Convert to PDF** (final): Print/export from Word or LibreOffice
3. **Verify text-searchable**: MiFILE requires text-searchable PDFs
4. **Check formatting**: Court captions, signatures, verification blocks
5. **File naming convention**: `[CaseNumber]_[DocumentType]_[Date].pdf`
