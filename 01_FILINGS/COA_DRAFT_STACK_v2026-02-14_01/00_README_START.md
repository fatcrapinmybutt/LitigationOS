# COA / JTC Draft Stack (Autodrafted) — 2026-02-14

This folder contains *court-draft* documents built from the currently pinned facts you’ve asserted in chat + the rule/vehicle structure needed for Michigan appellate original actions and related relief.

**Hard gates (why “final filing” isn’t automatic):**
1) **Operative order pin** (signed + file-stamped copy) + **entry date** + **service date**.
2) **Register of Actions / docket spine** proving what was filed/denied/continued and when.
3) **Record excerpts** (orders, transcripts, exhibits) supporting each proposition.
4) **Jurisdictional deadline check** (service method and holiday/weekend offsets).

The drafts below are written to be “plug-and-play” once record items are pinned. Missing record items are labeled **RECORD ITEM REQUIRED** (not invented).

## How to use with Gemini in IDE
1) Paste any signed/file-stamped orders/transcripts into `inputs/` (or point Gemini at them).
2) Ask Gemini to:
   - extract **entry/served/filed** dates,
   - extract **operative verbs** (“suspend”, “shall”, “continued until…”, “conditioned on…”),
   - produce **pinpoint citations** (page/paragraph).
3) Regenerate “Final” versions by replacing the **RECORD ITEM REQUIRED** blocks with pinned record.
