## IV. EXHAUSTION OF REMEDIES ANALYSIS

### Do You Need to Exhaust Administrative Remedies Before Filing in Court?

| Claim | Exhaustion Required? | Analysis |
|-------|---------------------|----------|
| **Fair Housing Act (42 USC § 3613)** | **NO** | Can file directly in federal court without exhausting HUD/MDCR |
| **Elliott-Larsen (MCL 37.2101)** | **PARTIAL** | Must file with MDCR or obtain right-to-sue letter before state court |
| **§ 1983 claims** | **NO** | Can file directly in federal court |
| **Mobile Home Commission Act** | **NO** | Can file LARA complaint AND state court action simultaneously |
| **CPS/MDHHS** | **DEPENDS** | Administrative remedy preferred but not strictly required for § 1983 |
| **Truth in Renting (MCL 554)** | **NO** | Private right of action, no exhaustion |
| **State tort claims** | **NO** | File directly in court |

### Key Takeaways
1. **Fair Housing Act**: No exhaustion required — filing HUD complaint is strategic, not mandatory
2. **Elliott-Larsen**: Should file MDCR complaint to preserve state claims (180-day deadline!)
3. **LARA**: Filing is supplemental; does not block court action
4. **MDHHS/CPS**: Administrative complaint builds record; § 1983 claims can proceed simultaneously

## VI. CROSS-AGENCY NOTIFICATION

When filing with multiple agencies, notify each of the other filings:

### Template Language
> "Complainant has also filed / intends to file complaints with the following agencies regarding related conduct: [LIST]. This notice is provided pursuant to HUD regulations and in the interest of coordination."

### Notification Matrix

| Filing With | Notify About |
|-------------|-------------|
| HUD | MDCR, LARA filings |
| MDCR | HUD, LARA filings |
| LARA | HUD, MDCR filings |
| MDHHS | All other filings (context) |

**HUD-MDCR Work-Share Agreement**: HUD and MDCR have a dual-filing agreement. Filing with one may automatically cross-file with the other. Verify at time of filing.
