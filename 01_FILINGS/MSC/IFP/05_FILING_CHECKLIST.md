## STEP 7 — AFTER FILING

### What to expect:

| Timeline | Event |
|----------|-------|
| **Day 0** | Filing received by MSC Clerk |
| **1-2 weeks** | Clerk processes filing; case number assigned (if new) |
| **2-4 weeks** | Court reviews IFP application |
| **4-8 weeks** | Court issues order granting or denying IFP |
| **If granted** | Clerk notifies you; proceed to file Application for Leave to Appeal |
| **If denied** | You may: (1) pay the fee, (2) request reconsideration with additional financial information, or (3) explore alternative relief |

### Important:
- [ ] **Monitor your mail** — Court orders are mailed to your address on file
- [ ] **Check TrueFiling** regularly if you filed electronically
- [ ] **Call the Clerk** at (517) 373-0120 if you haven't received a response within 6 weeks
- [ ] **Keep copies of everything** — all filed documents, proof of mailing, and any Court correspondence

### If IFP is GRANTED:
- [ ] File your Application for Leave to Appeal and/or Petition for Extraordinary Writs immediately
- [ ] The Application should already be prepared (or prepare it promptly)
- [ ] Reference the IFP order number in your filing
- [ ] Continue serving all documents on opposing counsel

### If IFP is DENIED:
- [ ] Read the Court's order carefully for the reason
- [ ] You may file a **Motion for Reconsideration** with additional financial information
- [ ] You may attempt to pay the $375.00 fee in installments (call Clerk to ask)
- [ ] Time limits for your underlying appeal still run — act promptly
