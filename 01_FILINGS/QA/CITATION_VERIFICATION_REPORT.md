## 📝 Methodology

1. **Extraction:** Regex patterns for MCR, MCL, MRE, Canon, Case Law, US/MI Constitution, FRCP, USC
2. **Verification:** Multi-strategy matching against `authority_master_index` (743 entries):
   - Exact identifier match
   - Base identifier match (without subsections)
   - LIKE search on identifier
   - Title search (for case law)
   - Partial party name match (for case law)
   - Fuzzy constitutional provision match
3. **Formatting check:** Cross-file consistency analysis
4. **Gap analysis:** Comparison against known important Michigan family law authorities

**Limitations:**
- Case law regex may produce false positives (Party v Party pattern)
- Subsection citations (e.g., MCR 2.003(C)(1)(a)) verified against base rule only
- Constitutional provisions may not all be in the master index
- Some authorities may be correct but absent from the database — "unverified" ≠ "wrong"
