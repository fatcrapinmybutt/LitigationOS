# JTC REQUEST FOR INVESTIGATION — REQUIREMENTS
## NOT a "Request for Investigation" — Citizens file a Request for Investigation (RFI)

### MANDATORY REQUIREMENTS (JTC RFI Form)
1. **NOTARIZATION** — Form must be signed before a notary public
2. **MAIL ONLY** — Do NOT submit by fax or email
3. **COPIES ONLY** — Attach copies of documents, NOT originals
4. **ADDRESS**: Judicial Tenure Commission, 3034 W. Grand Blvd., Suite 8-450, Detroit, MI 48202

### RFI CONTENTS
- Completed RFI Form (notarized)
- Narrative of judicial misconduct (detailed, factual)
- Exhibit list with descriptions
- Copies of all supporting documents
- Confidentiality note (if applicable — what you want kept confidential)

### CONFIDENTIALITY
- JTC proceedings are CONFIDENTIAL until formal charges filed
- MCR 9.205, 9.220 — Confidentiality of investigations
- Do NOT disclose filing of RFI publicly until JTC takes action

### PROCESS AFTER FILING
1. JTC receives RFI → screens for jurisdiction
2. If within jurisdiction → investigation opened
3. JTC investigates → may request response from judge
4. JTC decides: dismiss, admonish, or file Request for Investigation
5. Only JTC (not citizen) files Request for Investigation
6. If Request for Investigation → public hearing before Master
