## 🎓 LESSONS LEARNED

### Ollama Mistral Capabilities

**Strengths:**
- Excellent for Michigan legal research
- Accurate case citations and holdings
- Proper legal citation format
- Fast response times (60-90s per query)
- Local/private (no data sharing)

**Limitations:**
- Requires prompt refinement for JSON output
- Manual parsing of text responses needed
- Limited to training data cutoff
- Should verify all citations (standard practice)

**Best Practices:**
- Use specific jurisdiction in prompts
- Request structured output format
- Include example format in prompt
- Build in validation steps
- Cross-reference with authority databases

### PowerShell Automation

**Effective For:**
- Text processing and parsing
- File manipulation
- Report generation
- Ollama integration
- Batch processing

**Challenges:**
- PDF manipulation (requires external tools)
- Complex regex parsing
- String interpolation with special characters
