# JTC Complaint Packet (Structure)

## 1. Summary (tight)
- 6–10 bullets, each mapped to:
  - event date (or range)
  - order/transcript pinpoint
  - conduct theme (bias/procedure/record-access/etc.)

## 2. Chronology
- Use Chronology table (event → date → doc pinpoint).

## 3. Exhibit index
- “Copies, not originals” (see JTC packet instructions).
- Each exhibit labeled; include page pinpoints.

## 4. Attachments
- Operative orders
- Key transcripts (excerpts)
- Clerk notices / correspondence (if any)
