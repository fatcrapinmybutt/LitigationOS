# III. SERVICE METHODS — DETAILED INSTRUCTIONS

## A. Personal Service (MCR 2.105(A)) — PREFERRED

1. **Hire a process server.** Estimated cost: $50–$100 per defendant.
   - Muskegon County process servers (for local defendants): Search "process server Muskegon MI"
   - Wayne County process server (for Cox/Brown in Taylor): Search "process server Wayne County MI"
   - Out-of-state service (NJ, CO, NY): National service companies or local process servers in those jurisdictions

2. **Provide the process server with:**
   - Signed summons (MC 01) for each defendant
   - Copy of the First Amended Complaint with all exhibits
   - Service address and physical description (if known)
   - Special instructions (e.g., "Cox and Brown are at the same office — serve both in one trip")

3. **After service, the process server provides:**
   - Completed Proof of Service form (MC 04)
   - Date, time, and manner of service
   - Identification of person served

## B. Certified Mail (MCR 2.105(A)(2)) — ACCEPTABLE FOR LLCs

1. Send via **USPS Certified Mail, Return Receipt Requested, Restricted Delivery**
2. Include: Summons + Complaint + all exhibits
3. Keep the green card (return receipt) as proof of delivery
4. File Proof of Service (MC 04) with the green card attached

## C. Michigan Secretary of State — Long-Arm Service (MCL 600.715)

For foreign LLCs with no Michigan registered agent:

1. File with the Michigan Department of Licensing and Regulatory Affairs
2. Pay the required fee
3. LARA will forward the summons and complaint to the defendant at their registered address
4. This method satisfies due process for foreign entities transacting business in Michigan

# V. PROOF OF SERVICE REQUIREMENTS

For each defendant, file **MC 04 (Proof of Service)** with the court. Include:

1. Name of defendant served
2. Date and time of service
3. Method of service (personal / substitute / certified mail / SOS long-arm)
4. Address where served
5. Name of person who effected service
6. If substitute service: name and relationship of person accepting

**Deadline:** File Proof of Service within the time required by MCR 2.107 (before requesting default).

**Critical:** Defendants have **21 days after service** to file a responsive pleading (MCR 2.108(A)(1)). If they fail to respond, Plaintiff may move for **default** (MCR 2.603).
