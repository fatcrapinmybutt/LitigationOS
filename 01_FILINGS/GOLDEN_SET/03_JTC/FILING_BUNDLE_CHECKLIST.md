## JTC PROCESS OVERVIEW

The Judicial Tenure Commission handles complaints about judicial misconduct and disability.

### Stages of the JTC Process:

```
┌─────────────────────────────────────────────────────┐
│  STAGE 1: COMPLAINT FILED                           │
│  → Complaint received and reviewed by JTC staff     │
│  → Initial screening for jurisdiction and merit     │
│  → Timeline: 1–4 weeks for acknowledgment           │
├─────────────────────────────────────────────────────┤
│  STAGE 2: INVESTIGATION                             │
│  → JTC investigates allegations                     │
│  → May request additional information from          │
│    complainant                                      │
│  → May interview witnesses, review records          │
│  → Judge may be contacted for response              │
│  → Timeline: 2–12 months                            │
├─────────────────────────────────────────────────────┤
│  STAGE 3: FORMAL COMPLAINT (by JTC)                 │
│  → If merit found, JTC issues its own formal        │
│    complaint against the judge                      │
│  → Judge served with formal charges                 │
│  → Judge has right to respond                       │
│  → Timeline: Varies                                 │
├─────────────────────────────────────────────────────┤
│  STAGE 4: HEARING                                   │
│  → Public hearing before a master (retired judge)   │
│  → Both sides present evidence and testimony        │
│  → Master issues findings of fact and conclusions   │
│  → Timeline: 1–6 months after formal complaint      │
├─────────────────────────────────────────────────────┤
│  STAGE 5: RECOMMENDATION TO MICHIGAN SUPREME COURT  │
│  → JTC reviews master's report                      │
│  → JTC makes recommendation to MSC                  │
│  → Possible outcomes:                               │
│    • Dismissal                                      │
│    • Private censure                                │
│    • Public censure                                 │
│    • Suspension (with or without pay)               │
│    • Removal from office                            │
│    • Retirement (if disability)                     │
│  → MSC makes final decision                         │
│  → Timeline: 3–12 months after hearing              │
└─────────────────────────────────────────────────────┘
```

**Key Points:**
- Complaints are **confidential** during investigation (per MCR 9.261)
- Complainant may be contacted for additional information
- Judge is **not** immediately notified (investigation is confidential initially)
- The JTC decides whether to pursue formal charges — not the complainant
- Even if JTC dismisses, the complaint is on record

## POST-FILING NOTES

- JTC will send an **acknowledgment letter** (typically within 2–4 weeks)
- You may be contacted for additional information or clarification
- Investigation is **confidential** — do not discuss publicly
- You will be notified of the outcome when investigation concludes
- If JTC declines to act, this does NOT prevent other legal remedies (COA, MSC, etc.)
- Complaint remains on file even if dismissed — establishes pattern if future complaints arise
