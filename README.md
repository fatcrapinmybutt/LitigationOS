# LitigationOS

**Comprehensive Legal Knowledge Management System for Michigan Court Litigation**

---

## Overview

LitigationOS is a legally isolated, track-based legal knowledge management system.

## Repository Structure

- Core system files (TRACK_DEFINITIONS.json, DESIGN_SYSTEM.md, etc.)
- Six track directories (TRACK1-TRACK6)
- SCRIPTS directory with extraction and processing scripts

## Quick Start

See README.md in each track directory for track-specific information.
