## 🏦 What Is This?

The **Financial Intelligence & Market Analysis Starter Pack** is a free domain configuration pack for [CORTEX](https://fatcrapinmybutt.github.io/cortex-site/) — a local-first desktop intelligence platform that turns unstructured documents into structured, actionable intelligence.

This pack transforms CORTEX into a **financial crime detection and market surveillance workstation** capable of identifying securities fraud patterns, money laundering networks, insider trading signals, and regulatory compliance gaps — all from your local machine with **zero cloud dependency**.

Load financial filings, transaction records, emails, chat logs, compliance reports, or any text-based documents. CORTEX extracts entities, maps relationships, detects anomalies, and surfaces hidden patterns that would take a human analyst weeks to find.

> **🔐 100% Local Processing** — Your financial data never leaves your machine. No API keys. No cloud uploads. No third-party data exposure. Perfect for handling material non-public information (MNPI), client PII, and privileged compliance work.

## 📦 Installation

### Prerequisites
- [CORTEX Intelligence Platform](https://andrewpioneer6.gumroad.com/l/cjamvzo) (Pro or Bundle edition)

### Steps

```bash
# 1. Clone this starter pack
git clone https://github.com/fatcrapinmybutt/cortex-finance.git

# 2. Copy the domain pack into your CORTEX domains directory
cp cortex-finance/finance.json ~/.cortex/domains/

# 3. Launch CORTEX and select "Financial Intelligence" from the domain picker
```

Or manually: **Settings → Domain Packs → Import → Select `finance.json`**
