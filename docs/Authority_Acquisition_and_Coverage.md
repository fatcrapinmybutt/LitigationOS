# Authority Acquisition + Coverage Program

This build is *upload-only* (no web acquisition). To push toward comprehensive MI coverage:

## Target corpora
- Michigan Court Rules (MCR) full text (versioned)
- Michigan Rules of Evidence (MRE) full text
- Michigan Compiled Laws (MCL) targeted chapters (family law, PPO, contempt, mental health, records)
- Michigan Constitution (1963) + relevant provisions
- Published MSC + Published COA opinions (holdings extracted with pinpoints)
- Benchbooks + SCAO guidance + mandatory forms/instructions

## Coverage metrics (how you measure 'complete')
- MCR: rule-count coverage by section number
- MRE: rule-count coverage
- MCL: chapter/section coverage for a curated target list
- Caselaw: published-only coverage by year/court; holdings extracted
- Benchbook/forms: volume/edition coverage

## Crosswire minimums
Every proposition must have:
- Citation key (e.g., MCR 2.003)
- Pinpoint pointer (line-range or paragraph/section)
- Span text
- Source provenance (virtual_path + hash)
