### A. 14th Circuit Court — MiFILE E-Filing

**MiFILE** (Michigan's statewide e-filing system): https://mifile.courts.michigan.gov

| Step | Action |
|------|--------|
| 1 | Create MiFILE account at https://mifile.courts.michigan.gov |
| 2 | Select "14th Circuit Court — Muskegon County" |
| 3 | Select existing case OR file new case |
| 4 | Upload PDF documents (must be text-searchable, max 25MB per file) |
| 5 | Pay fees online (credit/debit) or upload fee waiver (MC 20) |
| 6 | System generates confirmation with timestamp |
| 7 | **You must STILL serve opposing parties separately** — MiFILE does NOT serve |

**Service on opposing parties (MCR 2.107):**
- **Represented party:** Serve attorney only (not party directly)
- **Pro se party:** Serve party directly
- **Methods:** First-class mail, personal delivery, email (if consented per MCR 2.107(C)(4))
- **Timing:** Same day as filing or next business day

### D. W.D. Michigan Federal — CM/ECF

**CM/ECF** (federal e-filing): https://ecf.miwd.uscourts.gov

| Step | Action |
|------|--------|
| 1 | Register for CM/ECF PACER account at https://pacer.uscourts.gov |
| 2 | Request e-filing privileges from W.D. Michigan Clerk |
| 3 | May need to file initial complaint on paper + request e-filing |
| 4 | Upload PDF documents via CM/ECF |
| 5 | Pay $405 filing fee or submit AO 240 (IFP) |
| 6 | **CM/ECF automatically serves all registered attorneys** |
| 7 | **Non-CM/ECF parties must be served traditionally** |

**Pro Se Note:** Federal courts may require pro se litigants to file paper copies initially. Call the Clerk: (269) 337-5652 (Kalamazoo) or (616) 456-2381 (Grand Rapids).

**Packet:** 07 — Federal §1983 Complaint

**Federal Service Requirements (FRCP 4):**
- Must serve each defendant within 90 days of filing
- Service options: US Marshal (free for IFP), process server, certified mail with waiver request
- **Waiver of service (FRCP 4(d)):** Send waiver request → defendant has 30 days to return → saves service costs → defendant gets 60 days (not 21) to answer
- If defendant refuses waiver → must personally serve → defendant pays service costs
