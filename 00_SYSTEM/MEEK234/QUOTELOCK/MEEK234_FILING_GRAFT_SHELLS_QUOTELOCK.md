# MEEK2 / MEEK3 / MEEK4 Filing Graft Shells (Quote-Lock Insertables)

## Use pattern
Drop each paragraph into: Facts/Record, Argument, Bias/Procedure, or Relief sections. Preserve quote-lock pin and attach underlying exhibit page.

## MEEK2 Custody / Parenting Time / Child Support
- **Record Recital Rebuttal Block:** Use `MEEK2_FINAL_EXHIBIT_GRADE_DROPINS.md` paragraphs MEEK2-### as the fact/rebuttal spine.
- **Procedural Hardening Rail:** Tie objections to MCR 3.215(D)-(F), MCL 552.507, MCR 2.107, MCR 2.517, and custody/PT hearing timing recitals from Chapter 3.
- **Relief asks:** restore parenting time; evidentiary hearing with live testimony; strike unsupported safety/mental-health recitals; findings on the record.

## MEEK3 PPO / Show-Cause / Contempt / Jailing
- **Record Recital Rebuttal Block:** Use `MEEK3_FINAL_EXHIBIT_GRADE_DROPINS.md` paragraphs MEEK3-###.
- **Procedural Hardening Rail:** Tie objections to MCR 3.707 + MCL 600.2950/600.2950a service/notice/modification framework; MCR 2.612 for fraud/misconduct/voidness posture where applicable.
- **Relief asks:** terminate/modify/vacate PPO and/or resulting tainted orders; set aside contempt/jailing consequences tied to parenting-time requests; evidentiary hearing on disputed factual predicates.

## MEEK4 Judge McNeill / Bias / Ex Parte USB / RUSCO / Martini / JTC-Higher Court
