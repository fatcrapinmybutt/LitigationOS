# Third-party components (free/open-source)

This runner is designed to be 100% free/open-source end-to-end.

## OCR
- Tesseract OCR — Apache 2.0 licensed. See upstream repo/license.  
- Leptonica (Tesseract dependency) — BSD-style.

## Vector DB
- Qdrant — run locally via Docker (self-hosted). See Qdrant docs.

## Embeddings
- Sentence-Transformers — open-source (model weights vary; defaults chosen are widely used open models).
- Optional: Ollama local embeddings via HTTP API.

You are responsible for complying with any model weight licenses you choose to use.
