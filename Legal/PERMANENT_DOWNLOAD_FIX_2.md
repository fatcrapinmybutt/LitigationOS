# Permanent “File Expired” Fix Strategy

Chat-delivered file links can expire depending on the platform/session.

This pack ships with:
1) a self-contained **world-first builder** script (`mbp_worldfirst_builder.py`) that reconstructs the full project folder locally and can export a ZIP any time; and
2) deterministic run folders + receipts in `MBP_OUT/RUN_*` for durable local outputs.

If a download link expires before you grab it, you can still recreate the project by running the builder script (copy-paste into a local `.py` file once).
