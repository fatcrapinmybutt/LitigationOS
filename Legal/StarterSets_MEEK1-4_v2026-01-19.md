# Starter Caselaw Sets (Verified-by-Source Links)

This file is a **starter** set intended to seed the LitigationOS caselaw curation pipeline (Section 21 of the spec). Each entry includes:

- Case name
- Citation (as reported)
- Domain proposition (why it matters)
- Source link

## MEEK2 — Custody / Parenting Time / Ex Parte Suspensions

### Core modification standards
1) **Vodvarka v Grasmeyer** — 259 Mich App 499; 675 NW2d 847 (2003)
   - Proposition: Defines “proper cause” / “change of circumstances” threshold for custody modification.
   - Source: https://www.courts.michigan.gov/siteassets/case-documents/uploads/opinions/final/coa/20030703_c236556_38_236556.opn.pdf

2) **Shade v Wright** — 291 Mich App 17; 805 NW2d 1 (2010)
   - Proposition: Clarifies that parenting-time modifications can have a different “proper cause/change of circumstances” analysis than custody changes; emphasizes child-focused inquiry.
   - Source: https://www.courts.michigan.gov/siteassets/case-documents/uploads/opinions/final/coa/20100720_c292388_49_292388.opn.pdf

### School / ECE / parenting-time-related standards (pipeline targets)

- **Lieberman v Orr** (Court of Appeals) — ECE / modification standards (verify and add pinpoints via pipeline).
- **Kaeb v Kaeb** (Court of Appeals) — parenting time & best-interests related (verify and add pinpoints via pipeline).
