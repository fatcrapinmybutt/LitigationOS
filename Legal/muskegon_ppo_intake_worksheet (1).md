# Muskegon PPO Intake Worksheet (Practitioner Use)

- Petitioner name / DOB / contact (confidential if needed)
- Respondent name / identifiers
- Relationship: Domestic / Non-Domestic / Stalking
- Immediate risk factors (weapons, recent threats, stalking behaviors)
- Requested relief (no-contact, no-stalking, firearms, residence, school, workplace)
- Ex parte requested? If yes, basis for immediate/irreparable harm
- Proposed order prepared? (Y/N)
- Service plan (personal/alternate) and deadlines
- LEIN entry follow-up calendar
- Next hearing date/time and witness list
- Evidence list (screenshots, messages, call logs, prior police reports)
