# PPO Filing — Muskegon (Family/14th Circuit)
- Select correct SCAO PPO form set (DV / Non‑DV / Stalking). Complete captions and safety details.
- If ex parte: file petition + proposed order; confirm clerk cut‑off; provide reliable contact info.
- Service: choose appropriate method; file proof; request confidential address measures if needed.
- At issuance: confirm LEIN entry; calendar return hearing; prepare exhibits.
- Media: coordinate under Media Protocol and AO 1989‑1 when safety/privacy is implicated.
