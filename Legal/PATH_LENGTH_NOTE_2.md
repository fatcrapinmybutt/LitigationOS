# PathLengthGuard note

This ZIP is a short-path edition to prevent Windows "path too long" extraction errors.

## Recommended extraction path
Extract to a shallow folder such as:
- `C:\L3pack`
- or `F:\L3pack`

## Mapping
- `PATH_MAP.csv` maps old (long) paths to new (short) paths.
- `START_HERE.md` provides the quick runbook.
