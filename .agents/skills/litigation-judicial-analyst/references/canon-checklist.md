# Canon Checklist — litigation-judicial-analyst

## Michigan Code of Judicial Conduct — Compliance Checklist

This checklist provides a systematic framework for evaluating judicial conduct
against the Michigan Code of Judicial Conduct. Use it to document Canon
violations in the 14th Judicial Circuit for recusal motions, JTC complaints,
and appellate record preservation.

---

## Canon 1: Integrity and Independence of the Judiciary

**Standard**: A judge should uphold the integrity and independence of the

```
================
Canon:          [1/2/3/4/5]
Sub-section:    [e.g., 3A.4]
Date observed:  [YYYY-MM-DD]
Case number:    [YYYY-NNNNNN-XX]
Judge:          [Name]

Description of conduct:
[Detailed factual description of what occurred]

Evidence sources:
- [Transcript, page X, line Y]
- [Docket entry dated YYYY-MM-DD]
- [Witness: Name, relationship to case]

Legal standard violated:
[Cite specific Canon language]

Severity assessment:
☐ MINOR (isolated, no harm to case outcome)
☐ MODERATE (pattern emerging, affected procedural fairness)
☐ SEVERE (clear violation, affected case outcome)
☐ EGREGIOUS (willful misconduct, requires formal action)

Recommended action:
☐ Document and monitor
☐ Include in recusal motion
☐ Include in JTC complaint
☐ Include in appellate record
```
