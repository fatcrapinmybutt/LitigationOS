## Data Collection Protocol

### For Each Hearing / Proceeding
1. Record date, time, duration
2. Note judge's tone toward each party (1–5 scale)
3. Count interruptions by party
4. Note any off-record comments
5. Record time allocated to each party for argument
6. Note any sua sponte actions
7. Record ruling and reasoning (if oral)
8. Note time between hearing and written order

### For Each Written Order
1. Date of order vs. date of hearing (delay analysis)
2. Whether order follows FOC recommendation (Lane A)
3. Whether order addresses all arguments raised
4. Quality of legal reasoning
5. Whether order is favorable / unfavorable and why
6. Comparison to similar orders in comparable cases

### For Aggregation
1. Maintain running statistics per judge
2. Compare against baseline (other 14th Circuit judges)
3. Update after every proceeding
4. Flag statistical anomalies for deeper analysis
