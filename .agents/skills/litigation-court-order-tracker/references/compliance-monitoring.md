# Compliance Monitoring — Michigan Court Order Reference

## Order Compliance Framework

### MCR 2.602 — Entry and Form of Judgments and Orders

Every order must meet MCR 2.602 requirements:
- **MCR 2.602(A)(1)**: Signed by the judge
- **MCR 2.602(A)(3)**: Filed with the clerk
- **MCR 2.602(B)(1)**: Effective upon entry (signing and filing)
- **MCR 2.602(B)(3)**: Must be in writing for enforcement purposes

### Order Catalog Structure

#### Order Entry Protocol
When a new order is entered in any lane:

1. **Obtain certified copy** from clerk within 7 days
2. **Extract provisions** — parse into individual obligations
3. **Assign compliance tracking** — who must do what, by when
4. **Check cross-lane conflicts** — does this conflict with orders in other lanes?
5. **Set deadlines** — calendar all compliance deadlines
6. **Link to evidence** — connect existing evidence to provision compliance/violation

#### Provision Extraction Template
```

  Case Number: _______________
  Lane: ____________________
```

### Compliance Status Categories

| Status | Definition | Action Required |
|--------|-----------|----------------|
| COMPLIED | Provision satisfied with documentation | Maintain evidence file |
| VIOLATED | Non-compliance documented with evidence | Contempt motion preparation |
| PENDING | Deadline not yet reached | Monitor deadline |
| PARTIALLY COMPLIED | Some but not all requirements met | Document gaps |
| DISPUTED | Parties disagree on compliance | Prepare for hearing |
| SUPERSEDED | Replaced by subsequent order | Archive with link to replacement |
| EXPIRED | Temporal provision no longer active | Verify expiration basis |
| STAYED | Suspended by appellate order | Monitor stay status |

### Cross-Lane Conflict Detection

#### Conflict Types
| Conflict | Example | Resolution |
|----------|---------|------------|
| **Direct contradiction** | PPO says no contact; custody order requires communication about exchanges | Motion to clarify/modify |
| **Temporal overlap** | Two orders set different parenting schedules for same period | Later order controls unless stayed |
| **Jurisdictional** | State and federal orders on same subject | Determine which court has jurisdiction |
| **Implicit** | Support order assumes custody arrangement that changed | Motion to modify support |

#### Conflict Resolution Priority
1. Constitutional rights prevail over statutory provisions
2. Federal court orders may preempt state orders (Supremacy Clause)
3. Later-entered orders supersede earlier orders on same subject
4. Specific provisions override general provisions
5. Appellate stays freeze lower court orders

### Monitoring Cadence

| Order Type | Review Frequency | Trigger for Review |
|-----------|-----------------|-------------------|
| Custody orders | Weekly | Parenting time exchanges |
| Support orders | Monthly | Payment dates |
| PPO orders | Daily | Contact/proximity |
| Discovery orders | Per deadline | Response due dates |
| Scheduling orders | Per event | Conference/hearing dates |
| Appellate orders | On entry | Stay/remand orders |

### Violation Documentation Protocol

For each detected violation:
```
VIOLATION REPORT

Order: [Case number, date, provision number]
Provision Text: [Exact quote]
Violation Date: [Specific date]
Description: [What happened vs. what was required]

EVIDENCE:
  Type: [Text message / Email / Photo / Witness / Calendar / Document]
  Description: [What the evidence shows]
  Preservation: [Screenshot saved / Original preserved / Witness identified]
  Location: [File path or evidence ID]

IMPACT:
  On L.D.W.: [Effect on child]
  On Petitioner: [Effect on non-violating party]
  Financial: [Costs incurred]

PRIOR NOTICE:
  Was respondent notified of obligation? [Yes/No + evidence]
  Was respondent warned about violation? [Yes/No + evidence]
  
PATTERN:
  Is this a repeat violation? [Yes/No]
  If yes, prior dates: [List]
  Escalation level: [First / Second / Pattern]

RECOMMENDED ACTION:
  [ ] Document only (isolated minor violation)
  [ ] Informal communication to opposing party
  [ ] Formal demand letter
  [ ] Contempt motion (MCR 3.606)
  [ ] Emergency motion (MCR 2.119(F))
```
