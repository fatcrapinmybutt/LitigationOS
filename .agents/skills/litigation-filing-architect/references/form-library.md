## Form Completion Checklists

### MC 290 — Domestic Relations Cover Sheet
- [ ] Case number in correct format (2024-001507-DC)
- [ ] Correct case type code (DC = domestic / custody)
- [ ] Both parties' full legal names
- [ ] Attorney or pro se designation
- [ ] Judge assignment (McNeill)
- [ ] Date filed

### FOC 10 — UCCJEA Jurisdiction Declaration
- [ ] Child's current state of residence
- [ ] Child's residence history (last 5 years)
- [ ] Prior custody proceedings in any state
- [ ] Persons with physical custody currently
- [ ] Statement of any pending proceedings
- [ ] Signed under penalty of perjury

### MC 01 — Civil Cover Sheet
- [ ] Case number (2025-002760-CZ)
- [ ] Case type code (CZ = civil general)
- [ ] All plaintiff and defendant names
- [ ] Amount in controversy
- [ ] Related cases (cross-reference Lane A if applicable)
- [ ] Attorney or pro se designation
- [ ] Judge assignment (Hoopes)

### CC 380 — Motion to Modify/Terminate PPO
- [ ] Original PPO case number (2023-5907-PP)
- [ ] Date of original PPO
- [ ] Specific modifications requested
- [ ] Factual basis for modification
- [ ] Whether respondent was served original PPO

## E-Filing Requirements — MiFILE / TrueFiling

### Document Format Requirements
| Requirement | Specification |
|-------------|--------------|
| File format | PDF (text-searchable preferred) |
| Max file size | 25 MB per document |
| File naming | Descriptive (e.g., "Motion_Modify_Custody_2025-06-15.pdf") |
| Confidential | Mark "Confidential" if contains protected info |
| Proposed orders | Uploaded in Word format (.docx) for judge editing |

### Common E-Filing Errors
1. **Wrong filing code**: Causes document to route to wrong clerk queue
2. **Oversized file**: Rejected at upload — split into multiple documents
3. **Non-searchable PDF**: Accepted but disadvantages your filing (judges search)
4. **Missing proposed order**: Motion accepted but processing delayed
5. **Wrong case number**: Document filed in wrong case entirely
