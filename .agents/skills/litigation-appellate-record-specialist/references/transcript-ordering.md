# Transcript Ordering — Michigan Court of Appeals Reference

## MCR 7.210(B) Transcript Requirements

### Ordering Timeline

| Event | Deadline | Authority |
|-------|----------|-----------|
| File claim of appeal | Day 0 | MCR 7.204(A) |
| Order transcripts | Within 14 days of claim | MCR 7.210(B)(1) |
| File proof of transcript order | With claim or within 14 days | MCR 7.210(B)(1) |
| Reporter completes transcript | 56 days (or as ordered) | MCR 7.210(B)(3)(a) |
| Brief due | 56 days after transcript filed | MCR 7.212(A)(1) |
| Motion for extension | BEFORE deadline expires | MCR 7.210(B)(3)(b) |

### Transcript Order Form Requirements

The transcript order must specify:
1. **Case number**: Both trial court and COA number (366810)
2. **Hearing dates**: Every date for which transcription is requested
3. **Type of proceeding**: Evidentiary hearing, motion, bench trial, etc.
4. **Court reporter name**: If known (check with clerk's office)
5. **Payment**: Deposit amount per court reporter's fee schedule
6. **Delivery**: Electronic and/or paper copies
7. **Expedited**: If needed for deadline compliance (costs more)

### Michigan Court Reporter Fee Schedule

Per MCR 8.108(B) and court reporter agreements:

| Service | Approximate Rate | Notes |
|---------|-----------------|-------|
| Original transcript | $3.00-$4.00/page | Varies by county |
| Copy | $0.50-$1.00/page | Additional parties |
| Expedited (7 days) | $5.00-$6.00/page | Must request in writing |
| Daily copy | $6.00-$8.00/page | Same-day delivery |
| Indigent rate | Reduced or waived | MCR 2.002 fee waiver |
| Electronic format | Same as paper | PDF preferred for e-filing |

### Identifying Required Transcripts

#### Step 1: Review Register of Actions
Extract every entry showing:
- "Hearing held" / "Hearing conducted"
- "Motion argued" / "Oral argument"
- "Trial" / "Evidentiary hearing"
- "Conference" (if on the record)
- "Ex parte" proceedings
- "FOC hearing" / "Referee hearing"

#### Step 2: Map Issues to Hearings
For each appellate issue, identify which hearing(s) are relevant:

| Appellate Issue | Relevant Hearing(s) | Transcript Priority |
|----------------|---------------------|-------------------|
| Custody determination error | Evidentiary hearing on best-interest factors | CRITICAL |
| Due process violation | Any hearing where violation occurred | CRITICAL |
| Evidentiary ruling error | Hearing where evidence was admitted/excluded | CRITICAL |
| Judicial bias | ALL hearings (pattern evidence) | HIGH |
| Support calculation error | Support hearing, FOC hearing | HIGH |
| Procedural irregularity | Scheduling conference, motion hearing | MEDIUM |

#### Step 3: Cost-Benefit Analysis
Not all transcripts are equally valuable. Prioritize:

**Must-Order (failure to order = issue forfeited):**
- Final evidentiary hearing on custody
- Any hearing where the complained-of ruling was made
- Hearing on motion that is subject of appeal

**Should-Order (strengthens appeal):**
- Preliminary hearings showing pattern of judicial conduct
- FOC hearings showing investigator approach
- Motion hearings with substantive legal argument

**May Defer (lower priority):**
- Scheduling conferences (usually administrative)
- Status conferences (usually off-record)
- Mediation sessions (usually confidential)

### Dealing with Transcript Delays

#### When Reporter is Late (MCR 7.210(B)(3))

1. **Document the delay**: Keep copies of all communications with reporter
2. **Notify the COA**: File a status report explaining the delay
3. **File motion for extension**: MCR 7.210(B)(3)(b) allows extensions when delay is not appellant's fault
4. **Request court intervention**: If reporter is significantly late, ask trial court to order compliance

#### When Transcript is Unavailable

If transcript cannot be obtained (recording failure, reporter unavailable):

Per MCR 7.210(B)(3):
1. Prepare **Statement of Proceedings** from best available sources:
   - Personal notes taken at hearing
   - Attorney/party notes
   - Court clerk's minutes
   - Any partial recording
2. Serve statement on opposing party
3. Opposing party has 14 days to object or propose amendments
4. Submit to trial court for settlement if disputed
5. Approved statement becomes part of the record

### Transcript Review Protocol

Once received, verify transcripts are complete:

- [ ] All requested hearing dates included
- [ ] Pages numbered consecutively per hearing
- [ ] Speaker identifications accurate (correct party names)
- [ ] L.D.W. referred to by initials only per MCR 8.119(H)
- [ ] Exhibits referenced in testimony can be located in court file
- [ ] Reporter's certification page present
- [ ] No "inaudible" gaps at critical moments
- [ ] Sealed portions properly handled (separate volume)

### Transcript Citation Format for Briefs

Per MCR 7.212(C)(7), cite transcripts as:
```
(Tr Vol [X], p [page])                          — single page
(Tr Vol [X], pp [start]-[end])                   — page range
(Tr [date], p [page])                            — when multiple hearing dates
(Evidentiary Hr'g Tr, [date], p [page])          — specific hearing type
(FOC Hr'g Tr, [date], p [page])                  — FOC hearing
(Bench Trial Tr, [date], p [page])               — bench trial
```

**Example citations for COA 366810:**
```
(Tr 01/15/2025, p 34)   — January 2025 custody hearing, page 34
(Tr Vol II, pp 45-48)   — Volume II, pages 45-48 of compiled transcript
(FOC Hr'g Tr, p 12)     — FOC hearing transcript, page 12
```

### Cost Management for Pro Se Litigants

1. **Fee waiver**: File MCR 2.002 affidavit of indigency for transcript costs
2. **Selective ordering**: Order only transcripts essential to appellate issues
3. **Statement of proceedings**: Use MCR 7.210(B)(3) for non-critical hearings
4. **Installment payments**: Some reporters accept payment plans
5. **County funding**: Some counties have transcript funds for indigent litigants
