# Record Compilation — Michigan Court of Appeals Reference

## MCR 7.210 Record on Appeal — Complete Guide

### What Constitutes the Record

Per MCR 7.210(A)(1), the record on appeal consists of:
1. **Original papers filed in the trial court** — all pleadings, motions, briefs, orders
2. **Transcript of proceedings** — all hearings, trials, and conferences on the record
3. **Exhibits** — all exhibits admitted into evidence at trial or hearing

### Record Compilation Checklist (COA 366810)

#### Phase 1: Document Inventory
- [ ] Obtain complete register of actions from 14th Circuit Court Clerk
- [ ] Cross-reference register against personal filing log
- [ ] Identify all hearings requiring transcripts
- [ ] Identify all exhibits admitted into evidence
- [ ] Identify sealed or confidential materials requiring special handling

#### Phase 2: Transcript Management
Per MCR 7.210(B)(1), appellant must order transcripts within 14 days:
- [ ] File transcript order form with court reporter
- [ ] Specify all hearing dates requiring transcription
- [ ] Pay transcript deposit (rates vary by county)
- [ ] Follow up weekly on transcript status
- [ ] If reporter delay, file motion for extension under MCR 7.210(B)(3)(b)

**Hearing types requiring transcripts:**
| Hearing Type | Priority | Status |
|-------------|----------|--------|
| Custody evidentiary hearing | CRITICAL | Must order immediately |
| PPO hearing | HIGH | Required for Lane D issues |
| Motion hearing (oral argument) | HIGH | Captures court's reasoning |
| Settlement/mediation | USUALLY EXEMPT | Check if on record |
| Ex parte proceeding | CRITICAL | Often not transcribed — verify |
| FOC hearing | HIGH | May not be transcribed by default |

#### Phase 3: Exhibit Collection
- [ ] Obtain list of admitted exhibits from trial court file
- [ ] Request copies of all admitted exhibits
- [ ] Verify exhibit numbers match register of actions
- [ ] Organize exhibits in admission order
- [ ] Apply Bates numbers for appendix reference

**Exhibit Categories (Lane A — Custody):**
| Category | Typical Exhibits | Handling |
|----------|-----------------|----------|
| Court orders | Custody orders, PPO, support orders | Certified copies required |
| Evaluations | Custody evaluation, GAL report | Sealed — motion needed |
| Financial | Tax returns, pay stubs, bank records | Verify admission status |
| Communications | Emails, texts, voicemails | Authentication required |
| Medical | Child's medical records | HIPAA-sensitive handling |
| School | Report cards, attendance, IEP | May require subpoena for certified copies |
| Police | Reports, 911 records, CAD logs | FOIA if not in court file |
| Photographs | Dated and authenticated photos | EXIF metadata preservation |

#### Phase 4: Record Organization

**Recommended Organization Scheme:**
```
RECORD ON APPEAL — COA 366810 (Pigors v Watson)
├── Volume I:   Register of Actions (certified)
├── Volume II:  Pleadings and Motions (chronological)
├── Volume III: Court Orders (chronological)
├── Volume IV:  Transcripts
│   ├── Hearing Date 1 — [description]
│   ├── Hearing Date 2 — [description]
│   └── ...
├── Volume V:   Exhibits (Bates numbered)
│   ├── Exhibit 1 — [description] (Bates 001-015)
│   ├── Exhibit 2 — [description] (Bates 016-022)
│   └── ...
└── Volume VI:  Sealed Materials (under separate cover)
```

#### Phase 5: Certification
- [ ] Submit compiled record to trial court clerk for certification
- [ ] Clerk stamps and certifies the record as complete
- [ ] File certified record with COA
- [ ] Serve copy on opposing party (or notice of filing)

### MCR 7.210(B)(3) — Statement of Proceedings

When transcript is unavailable:
1. Appellant prepares statement of proceedings from best available sources
2. Statement served on opposing party
3. Opposing party has 14 days to object or propose amendments
4. If no objection, statement becomes part of the record
5. If dispute, trial court settles the statement

**When to use statement of proceedings:**
- Court reporter deceased or unavailable
- Recording equipment malfunction
- Hearing not recorded (some FOC proceedings)
- Cost of transcript prohibitive (file with affidavit of indigency)

### Record Completeness Standards

The record must support every issue raised on appeal:

| Issue | Required Record |
|-------|----------------|
| Factual findings error | Transcript + exhibits reviewed by court |
| Legal error | Motion/brief + court's ruling + any oral reasoning |
| Procedural error | Register of actions + relevant motions/orders |
| Evidentiary error | Transcript showing objection + ruling + offer of proof |
| Due process | Full transcript + all notices/service proofs |
| Judicial bias | Transcript showing complained-of conduct |

### Common Errors in Record Compilation

1. **Assuming electronic filing = complete record** — e-filed documents may not match the official court file
2. **Omitting minute orders** — brief oral rulings may not generate written orders but are in the register
3. **Missing confidential supplements** — sealed documents require separate handling
4. **Incorrect date ranges** — companion case (2023-5907-PP) documents may be relevant to 2024-001507-DC appeal
5. **Excluding companion case records** — if issues span both case numbers, both records may be needed
