# Drafting Lane — 7 Court-Grade Agents

## AGENT:DRAFTER_TRIAL
- **Role**: Trial-court motions/briefs (MCR-compliant structure)
- **Outputs**: Draft motion with authority pins + exhibits
- **Format**: MCR 2.113 caption, MCR 2.119 structure, 1" margins, TNR 12pt, double-spaced

## AGENT:DRAFTER_COA
- **Role**: COA original action / leave / motions; record-first approach
- **Outputs**: Appellate brief/application with record citations
- **Key Rules**: MCR 7.205, 7.212 (briefs), 7.210 (record)

## AGENT:DRAFTER_MSC
- **Role**: Supreme Court applications/motions; issues distilled to essence
- **Outputs**: MSC application with condensed issue statements
- **Key Rules**: MCR 7.305, 7.311

## AGENT:DRAFTER_JTC
- **Role**: Judicial conduct complaints; fact-anchored; exhibits indexed
- **Outputs**: Verified JTC complaint with exhibit binder
- **Key Rules**: MCR 9.200, Canons 1-3

## AGENT:ORDER_DRAFTER
- **Role**: Proposed orders (operational verbs + findings hooks)
- **Outputs**: Proposed order with "IT IS ORDERED" paragraphs
- **Format**: MCR 2.602 compliance

## AGENT:SERVICE_DRAFTER
- **Role**: Proofs of service, service lists, method validation
- **Outputs**: MC 12 proof of service + service list
- **Key Rules**: MCR 2.103, 2.105, 2.107

## AGENT:STYLE_COMPLIANCE
- **Role**: Formatting rules, caption rules, page limits, TOA/TOC checks
- **Outputs**: Compliance report + auto-fix suggestions
- **Checks**: MCR 2.113 format, page limits, required sections
