# Lane C: Convergence / County / Judges — 7 Legal Actions

> Lane C exists ONLY for claims where judicial or government misconduct spans BOTH
> the custody case (Lane A) AND the housing case (Lane B).

## Convergence Adversaries

| Entity | Why Convergence | Pattern |
|--------|----------------|---------|
| **Muskegon County** | Governmental entity hosting both courts | Systemic issues |
| **14th Judicial Circuit** | Court system itself | Administrative patterns |
| **Hon. Jenny L. McNeill** | Custody case judge — bias + misconduct | Custody + county |
| **Hon. Hoopes** | Housing case judge (if misconduct found) | Housing + county |
| **FOC / Pamela Rusco** | State actor in custody — county employee | Government accountability |

## Legal Actions

| # | Action | Authority | Basis |
|---|--------|-----------|-------|
| C1 | §1983 Monell County Liability | 42 USC §1983, Monell v. Dept of Social Services | County policy/custom caused constitutional harm |
| C2 | §1983 Pattern of Judicial Misconduct | 42 USC §1983, 14th Amdt | Systemic due process violations across cases |
| C3 | JTC Complaint (Cross-Case Pattern) | MCR 9.200 | McNeill misconduct visible in BOTH case lanes |
| C4 | MSC Administrative Complaint | MCR 9.100+ | 14th Circuit systemic issues |
| C5 | Federal Civil Rights (Consolidated) | 42 USC §1983, §1985 | Conspiracy spanning custody + housing to deprive rights |
| C6 | Mandamus (County-Level) | MCR 3.305 | Compel county to provide constitutional due process |
| C7 | Taxpayer Action | MCL 211.1+ | County waste/misconduct with public funds |

## Filing Priority (Recommended Order)

1. **JTC cross-case pattern** (C3) — Strongest when both lanes have established records
2. **§1983 Monell** (C1) — County liability requires showing custom/policy
3. **MSC administrative** (C4) — Systemic issues at circuit level
4. **Federal consolidated** (C5) — §1983 + §1985 conspiracy
5. **Mandamus** (C6) — Compel due process
6. **Pattern of misconduct** (C2) — Ongoing violations
7. **Taxpayer action** (C7) — If public funds misused

**NOTE:** Lane C filings should generally come AFTER initial Lane A and Lane B filings
establish the record. You need documented misconduct in both lanes before convergence claims
have maximum force.

## Convergence Rules (Explicit)

### ✅ CAN Do

- Cite McNeill's custody misconduct to show county-wide pattern
- Cite housing harms alongside custody harms in §1983 against county
- Aggregate damages across BOTH case lanes for federal claims
- Reference both judges' conduct when showing 14th Circuit systemic issues
- Use FOC misconduct (Lane A) as evidence of county employee pattern (Lane C)

### ❌ CANNOT Do

- Merge case numbers (each case retains its own number)
- File housing evidence in custody case (Lane B → Lane A mixing)
- File custody evidence in housing case (Lane A → Lane B mixing)
- Claim convergence without documented misconduct in BOTH lanes
- Use Lane C as a shortcut to avoid proper lane separation

## Evidence Scoring for Lane C

Lane C actions receive **bonus scoring** from both Lane A and Lane B evidence:

```
lane_c_score = (lane_a_evidence × 0.5 + lane_b_evidence × 0.5 + lane_c_specific × 1.0)
               × authority_backing × pattern_strength
```

Where `pattern_strength` = number of distinct misconduct instances spanning both lanes.
