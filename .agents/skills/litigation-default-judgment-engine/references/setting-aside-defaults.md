# Setting Aside Defaults — Michigan Reference

## MCR 2.603(D) Framework

### Two Distinct Standards

| Stage | What's Being Set Aside | Standard | Key Requirement |
|-------|----------------------|----------|-----------------|
| Pre-judgment | Entry of default | Good cause | Show excusable neglect or mistake |
| Post-judgment | Default judgment | Good cause + meritorious defense affidavit | Show both good cause AND viable defense |

### "Good Cause" Analysis

Courts evaluate the totality of circumstances. No single factor is dispositive.

#### Factor 1: Whether Default Was Willful
**Weighs FOR setting aside when:**
- Defendant didn't receive actual notice of the lawsuit
- Service was technically proper but defendant was traveling, hospitalized, or incapacitated
- Defendant's attorney made a calendaring error
- Defendant misunderstood procedural requirements (especially pro se)

**Weighs AGAINST setting aside when:**
- Defendant deliberately ignored the complaint
- Defendant told others they weren't going to respond
- Multiple contacts/notices were ignored
- Defendant is sophisticated or represented by counsel

#### Factor 2: Extent of Prejudice to Non-Defaulting Party
**Prejudice analysis:**
- Loss of evidence (witnesses unavailable, documents destroyed)
- Statute of limitations has run on related claims
- Tactical disadvantage from delay
- Costs incurred in obtaining default

**NOT prejudice:**
- Having to litigate on the merits (this is expected)
- Delay alone (without specific harm)
- Loss of "windfall" default judgment

#### Factor 3: Whether Meritorious Defense Exists
**Standard**: The proposed defense need not be certain to prevail — it must be "meritorious" in the sense that it could succeed if proven.

**Sufficient meritorious defenses:**
- Factual dispute on essential element of plaintiff's claim
- Affirmative defense (statute of limitations, consent, accord)
- Counterclaim that would offset damages
- Challenge to damages calculation

**Insufficient defenses:**
- Conclusory denial without facts
- Defense that is legally insufficient as a matter of law
- Defense contradicted by undisputed evidence

### Motion to Set Aside — Requirements

#### For Entry of Default (MCR 2.603(D)(1))
```
MOTION TO SET ASIDE ENTRY OF DEFAULT

Required contents:
1. Case caption and case number
2. Date default was entered
3. Explanation of why default was entered (good cause)
4. Proposed responsive pleading (attached)
5. Statement of meritorious defense (recommended but not required)
6. Request for relief

Supporting documents:
□ Affidavit explaining circumstances of default
□ Proposed answer/responsive pleading
□ Evidence supporting good cause (medical records, travel docs, etc.)
□ Proof of service on all parties
```

#### For Default Judgment (MCR 2.603(D)(2))
```
MOTION TO SET ASIDE DEFAULT JUDGMENT

Required contents (in addition to above):
1. All items required for entry of default
2. AFFIDAVIT OF MERITORIOUS DEFENSE (mandatory)
   - Specific facts constituting the defense
   - Sworn statement, not conclusory
   - Sufficient detail to show defense could prevail
3. Explanation of delay in moving to set aside

Additional supporting documents:
□ Affidavit of meritorious defense (detailed, sworn)
□ Documentary evidence supporting the defense
□ Timeline showing diligence after discovering default
□ Proposed answer with affirmative defenses
```

### Timing Considerations

| Situation | Timing | Authority |
|-----------|--------|-----------|
| Motion to set aside default | Before judgment — any time | MCR 2.603(D)(1) |
| Motion to set aside default judgment | "Reasonable time" — no fixed deadline | MCR 2.603(D)(2) |
| Independent action for relief | Within 1 year, or longer for void judgment | MCR 2.612(C) |
| Void judgment (defective service) | No time limit | Due process |

### Court's Discretion

The trial court has broad discretion:
- **Liberal policy**: Michigan courts strongly prefer merit adjudication
- *Shawl v Spence*, 236 Mich App 120, 126 (1999): "doubts should be resolved in favor of hearing on the merits"
- Appellate review: abuse of discretion standard
- Reversals are common when trial court refuses to set aside a default where good cause exists

### Opposing a Motion to Set Aside

If you obtained the default, here's how to oppose the motion:

1. **Challenge good cause**: Show the default was willful, not excusable
2. **Show prejudice**: Document specific harm from the delay
3. **Attack meritorious defense**: Show proposed defense is conclusory or legally insufficient
4. **Emphasize delay**: If motion was filed long after default, argue laches
5. **Document costs**: Show wasted resources from default proceedings

### Response Template (Opposing Set-Aside)
```
RESPONSE TO MOTION TO SET ASIDE DEFAULT [JUDGMENT]

I. PROCEDURAL HISTORY
   - Date of service: _______
   - Response deadline: _______
   - Date default entered: _______
   - Date of this motion: _______ (delay: ____ days)

II. DEFAULT WAS WILLFUL
   - [Evidence that defendant knew of the case]
   - [Evidence of deliberate non-response]

III. PREJUDICE TO PLAINTIFF
   - [Specific evidence lost or degraded]
   - [Costs incurred in obtaining default]
   - [Other concrete harm from delay]

IV. NO MERITORIOUS DEFENSE
   - [Why proposed defense is insufficient]
   - [Undisputed evidence defeating the defense]

V. CONCLUSION
   Plaintiff respectfully requests the Court deny Defendant's
   motion and maintain the default [judgment].
```

## Practical Considerations for Pigors v Watson

### As Plaintiff (Pursuing Default)
- Verify service is PERFECT before requesting default — any defect is fatal
- Prepare for prove-up hearing as if it were trial — court determines damages
- Be ready for set-aside motion — it's almost always filed
- File SCRA affidavit with EVERY default application

### As Defendant (Defending Against Default)
- Respond IMMEDIATELY to any complaint — don't wait until the last day
- If you've been defaulted, file motion to set aside IMMEDIATELY
- Prepare strong meritorious defense affidavit — specific facts, not conclusions
- Demonstrate diligence — the faster you move, the better your chances
