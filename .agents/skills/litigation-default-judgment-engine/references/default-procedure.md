# Default Judgment Procedure — MCR 2.603 Reference

## Overview

Default judgment is obtained when a defendant fails to respond to a complaint within the time allowed. Michigan uses a two-step process: entry of default (ministerial) followed by default judgment (judicial).

## Step 1: Entry of Default — MCR 2.603(A)

### Prerequisites
1. Complaint properly filed and served
2. Service complies with MCR 2.105 (personal), MCR 2.106 (substituted), or court order
3. Response deadline has expired:
   - Personal service: 21 days from service date
   - Service by mail: 21 days + 3 days (MCR 2.107(C)(3))
   - Service by publication: per court order
4. No responsive pleading filed by defendant

### Filing Requirements
| Document | Required? | Notes |
|----------|-----------|-------|
| Request for Entry of Default | YES | SCAO form MC 03 |
| Proof of Service | YES | Affidavit of service with details |
| Military Affidavit (SCRA) | YES | Non-military status verification |
| Copy of Complaint | RECOMMENDED | For clerk reference |

### Clerk's Entry
- Clerk enters default as ministerial act — no hearing required
- Default establishes LIABILITY only
- Does NOT establish damages amount (unless liquidated)
- Defendant may no longer file responsive pleading without court order

## Step 2: Default Judgment — MCR 2.603(B)

### For Liquidated (Certain Sum) Claims
If the claim is for a sum certain (e.g., unpaid loan of specific amount):
- Clerk may enter judgment without hearing
- Requires affidavit of amount due
- Must include prejudgment interest calculation if requested

### For Unliquidated (Uncertain) Claims
If damages must be determined (most litigation claims):
- Court hearing REQUIRED — called "prove-up hearing"
- Plaintiff must present evidence of damages
- Court determines amount — may be less than requested
- Sworn testimony often required

### Prove-Up Hearing Preparation
```
PROVE-UP HEARING CHECKLIST

1. EVIDENCE OF DAMAGES:
   □ Economic damages documentation (receipts, bills, records)
   □ Non-economic damages testimony (impact statements)
   □ Expert testimony (if applicable — damages expert)
   □ Calculation worksheet (detailed, line-by-line)

2. LEGAL AUTHORITY:
   □ Damages statute citations (MCL 600.6013, etc.)
   □ Case law supporting damage categories
   □ Statutory multiplier authority (if applicable)

3. PROCEDURAL DOCUMENTS:
   □ Proposed judgment for court signature
   □ SCRA affidavit (if not already filed)
   □ Proof of notice to defendant (default parties get notice of hearing)

4. TESTIMONY PREPARATION:
   □ Plaintiff testimony outline
   □ Expert witness preparation (if applicable)
   □ Exhibits organized and marked
```

## MCR 2.603(D) — Setting Aside Default

### Standard: "Good Cause"
For entry of default (before judgment):
- Must show "good cause" — liberal standard
- Factors: (1) whether default was willful, (2) extent of prejudice to non-defaulting party, (3) whether meritorious defense exists
- Courts STRONGLY prefer merit adjudication

### Standard: "Good Cause" + Meritorious Defense Affidavit
For default judgment (after judgment entered):
- Must show "good cause" AND file affidavit of meritorious defense
- Meritorious defense: facts that, if proven, would constitute a complete or partial defense
- Motion must be filed within reasonable time

### Factors Courts Consider
| Factor | Favors Setting Aside | Favors Maintaining Default |
|--------|---------------------|---------------------------|
| Willfulness | Excusable neglect, mistake | Deliberate avoidance |
| Delay | Prompt motion after discovery | Months of inaction |
| Prejudice | No prejudice to plaintiff | Significant prejudice |
| Meritorious defense | Strong factual defense | No viable defense |
| Amount at stake | Large judgment | Small amount |
| Public interest | Resolution on merits | Finality of judgments |

### Key Case Law
- *Shawl v Spence*, 236 Mich App 120, 126 (1999) — "doubts should be resolved in favor of hearing on the merits"
- *Alken-Ziegler v Waterbury Headers*, 461 Mich 219 (1999) — liberal set-aside policy
- *Huntington Nat'l Bank v Ristich*, 292 Mich App 376 (2011) — good cause analysis

## Service Requirements for Default

### MCR 2.105 — Personal Service
- Process served personally on defendant
- Must identify defendant and deliver copy of summons and complaint
- Server must be 18+ and not a party

### MCR 2.106 — Substituted Service
If personal service cannot be made after reasonable diligence:
- Service at defendant's dwelling with person of suitable age
- Service by registered mail (with signed receipt)
- Motion for alternate service under MCR 2.105(J)

### MCR 2.108 — Service on Specific Entities
- Corporations: registered agent or officer
- LLCs: registered agent or member/manager
- Government: designated agent per statute

## SCRA Compliance (50 USC § 3931)

### Requirement
Before ANY default judgment, plaintiff must file affidavit stating:
1. Defendant IS a member of the military (requires appointment of attorney), OR
2. Defendant IS NOT a member of the military, OR
3. Plaintiff is unable to determine military status

### Verification
- Check DoD Manpower Data Center: https://scra.dmdc.osd.mil/
- Document search results
- File affidavit with search documentation

### Penalty for Non-Compliance
- Judgment may be reopened within 60 days of release from military
- Possible fine or imprisonment for false affidavit
- Judgment is VOIDABLE (not void) — defendant must move to set aside

## Timeline Summary

```
Day 0:   Complaint filed and served
Day 21:  Response deadline (personal service)
Day 22+: May request entry of default (MCR 2.603(A))
Day 28+: Court schedules prove-up hearing (unliquidated)
Day 35+: Prove-up hearing → Default judgment entered
Day 36+: Defendant may move to set aside (MCR 2.603(D))
```
