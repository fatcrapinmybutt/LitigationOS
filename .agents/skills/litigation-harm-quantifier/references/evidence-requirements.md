# Evidence Requirements — Damages Proof Matrix

Complete mapping of what evidence is required to support each damages category in Pigors v Watson. For each category: minimum evidence, ideal evidence, expert requirements, and documentation best practices.

## I. Evidence Standards

### Michigan Burden of Proof for Damages

- **Standard**: Preponderance of the evidence (more likely than not)
- **Certainty**: Damages must be proven with "reasonable certainty" — not mathematical precision, but more than speculation
- **Authority**: Brinker v Hobbs, 388 Mich 655 (1972); Hofmann v Auto Club Ins Assn, 211 Mich App 55 (1995)

### Evidence Hierarchy (Strongest to Weakest)

1. **Official records**: Court orders, government documents, certified copies
2. **Third-party business records**: Medical records, bank statements, employment records
3. **Expert testimony**: Professional opinions with methodology
4. **Documentary evidence**: Contracts, leases, photographs, communications
5. **Third-party witness testimony**: Non-party observations
6. **Plaintiff testimony**: Direct account (necessary but insufficient alone for most categories)

## II. Economic Damages Evidence Matrix

### A. Lost Wages

|---|---|---|---|
| **Minimum Required** | | | |

## III. Non-Economic Damages Evidence Matrix

### A. Emotional Distress

| Evidence Type | Description | Importance | Notes |
|---|---|---|---|
| **Minimum Required** | | | |
| Plaintiff declaration/testimony | Describe nature, severity, duration of distress | Essential | Specific, detailed, not conclusory |
| **Strongly Recommended** | | | |
| Therapist/counselor records | Professional documentation of mental state | Very High | Contemporaneous records are most credible |
| Medication records | Prescriptions for anxiety, depression, insomnia | High | Shows severity requiring medical intervention |
| Corroborating witness testimony | Family/friends observing behavioral changes | High | Overcomes "self-serving" objection |
| Journal/diary entries | Contemporaneous recording of emotional state | Medium | Must be genuine, not litigation-prepared |
| **Physical Manifestation Evidence** | | | |
| Medical records (physical symptoms) | Insomnia, weight change, headaches, GI issues | Very High | Strengthens claim significantly |
| Work performance records | Declining performance, absences | Medium | Shows functional impairment |
| Photos showing physical changes | Weight loss/gain, hair loss, aging effects | Medium | Powerful visual evidence |
| **Expert Required If** | | | |
| PTSD or clinical diagnosis claimed | Psychologist or psychiatrist evaluation | Essential | Must meet Daubert/MRE 702 standard |
| Future emotional harm projected | Mental health professional opinion | Essential | Expert on prognosis and treatment needs |

| Evidence Type | Description | Importance | Notes |
|---|---|---|---|
| **Minimum Required** | | | |
| Court-ordered parenting schedule | Baseline for what was supposed to happen | Essential | Certified copy of custody order |
| Parenting time log | Record of actual vs. ordered time | Essential | Date, scheduled time, what actually happened |

## IV. Punitive Damages Evidence

| Evidence Type | Purpose | Notes |
|---|---|---|
| Prior similar conduct | Shows pattern, not isolated incident | Other lawsuits, complaints, regulatory actions |
| Defendant's knowledge | Knew conduct was harmful | Warnings, complaints, professional standards |
| Defendant's motive | Acted with malice or improper purpose | Communications revealing intent |
| Financial condition of defendant | Relevant to amount needed for deterrence | Required for punitive damages amount |
| Industry standards | Shows defendant deviated from norms | Professional standards, regulations |

## V. Documentation Best Practices

### Real-Time Documentation Rules

1. **Document immediately**: Record events the same day they occur
2. **Be specific**: Dates, times, exact quotes, names of witnesses
3. **Preserve originals**: Keep original documents; provide copies to court
4. **Photograph everything**: Housing conditions, physical evidence, injuries
5. **Screenshot digital evidence**: Text messages, emails, social media — with metadata visible
6. **Create a running log**: Maintain chronological incident log updated regularly
7. **Back up digitally**: Cloud storage + local copy + paper copy (triple redundancy)
8. **Don't alter**: Never edit, redact, or modify evidence after creation (except privilege redaction with log)

### Evidence Organization System

```
EVIDENCE MASTER FILE
├── 01_COURT_ORDERS/         ← All orders, judgments, signed documents
├── 02_COMMUNICATIONS/       ← Texts, emails, letters (by date)
├── 03_FINANCIAL/            ← Pay stubs, receipts, bills, bank records
├── 04_MEDICAL/              ← Medical records, therapy notes, prescriptions
├── 05_HOUSING/              ← Lease, inspection reports, photos, complaints
├── 06_CUSTODY/              ← Parenting time logs, exchange documentation
├── 07_PHOTOS_VIDEO/         ← All visual evidence (timestamped)
├── 08_WITNESS_STATEMENTS/   ← Declarations, affidavits, contact info
├── 09_EXPERT_REPORTS/       ← Any expert opinions obtained
└── 10_RESEARCH/             ← Comparable cases, legal research
```

### Chain of Custody for Digital Evidence

1. Take screenshot with date/time visible on device
2. Save to cloud storage immediately (creates timestamp)
3. Print hard copy and note date printed
4. For text messages: photograph the phone screen showing the message thread with contact info visible
5. For emails: save as PDF with full headers
6. For social media: capture URL, profile information, and content in single screenshot
7. Consider using a notary for critical digital evidence (sworn printout)
