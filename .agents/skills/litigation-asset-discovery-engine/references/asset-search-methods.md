# Asset Search Methods — Michigan Family Law Reference

## Formal Discovery Tools

### MCR 2.309 — Interrogatories

#### Financial Interrogatory Templates

**Interrogatory Set 1: Income Identification**
```
1. State your current employer, position, and gross annual income.
2. List all sources of income for the past 3 tax years including:
   (a) Wages, salaries, bonuses, commissions
   (b) Self-employment income
   (c) Rental income
   (d) Investment income (interest, dividends, capital gains)
   (e) Trust distributions
   (f) Government benefits (disability, unemployment, social security)
   (g) Any other income source
3. For each source identified in #2, state the annual amount
   received for each of the last 3 calendar years.
4. If you are currently unemployed or underemployed, state:
   (a) Date employment ceased or hours reduced
   (b) Reason for cessation/reduction
   (c) Efforts to obtain comparable employment
   (d) Your education level and professional certifications
```

**Interrogatory Set 2: Asset Identification**
```
5. List every bank account (checking, savings, money market, CD)
   in which you have or had an interest in the last 3 years:
   (a) Institution name and branch
   (b) Account number (last 4 digits)
   (c) Account type
   (d) Current balance
   (e) Names of all account holders
6. List every investment or brokerage account:
   (a) Institution name
   (b) Account type (IRA, 401k, brokerage, HSA)
   (c) Current value
   (d) Account holder names
7. List all real property in which you have any interest:
   (a) Address
   (b) Type of interest (ownership, lease, beneficial)
   (c) Current fair market value
   (d) Mortgage balance
   (e) Entity through which property is held, if any
8. List all vehicles, boats, or recreational vehicles you own or lease.
9. List all business interests including:
   (a) Entity name and type (LLC, Corp, Partnership)
   (b) Percentage ownership
   (c) Current estimated value
   (d) Role in entity
```

### MCR 2.310 — Requests for Production

#### Financial Document Requests
```
1. Federal and state tax returns (complete with all schedules,
   W-2s, and 1099s) for the past 3 tax years.
2. Pay stubs for the past 12 months from all employers.
3. Bank statements for all accounts for the past 24 months.
4. Brokerage/investment account statements for past 24 months.
5. Retirement account statements (401k, IRA, pension) for past 24 months.
6. Credit card statements for past 24 months.
7. Loan applications submitted in the past 3 years.
8. Business financial statements (P&L, balance sheet) if self-employed.
9. Real property closing documents for any transactions in past 5 years.
10. Insurance policies (life, disability, property) with declarations pages.
```

### MCR 2.305 — Subpoenas to Third Parties

#### Bank Subpoena Protocol
1. Identify institution from interrogatory responses or other sources
2. Prepare subpoena duces tecum under MCR 2.305(A)
3. Specify: account statements, signature cards, safe deposit records
4. Serve on institution's registered agent or custodian of records
5. Serve copy on opposing party per MCR 2.305(A)(3)
6. Allow 14 days for compliance
7. Cost: $25-50 service fee per institution

#### Employer Subpoena
- Wage and salary verification
- Bonus/commission history
- Benefits valuation (health, retirement, stock options)
- Employment agreement terms

## Dissipation Tracking

### *Sparks v Sparks* Framework (440 Mich 141, 158 (1992))

Elements to establish dissipation:
1. **Asset existed**: Documentary proof the asset was in marital estate
2. **Asset was dissipated**: Transfers, expenditures, or destruction
3. **Timing**: During or in contemplation of divorce proceedings
4. **Purpose**: Not for legitimate marital purpose

### Transaction Pattern Analysis
```
RED FLAGS for dissipation:
- Large withdrawals near separation date
- Transfers to family members or new partners
- Unusual luxury purchases during contested proceedings
- Cash withdrawals with no documented purpose
- New accounts opened at unfamiliar institutions
- Cryptocurrency purchases (increasingly common)
- Overpayment of taxes or creditors (parking money)
- "Gifts" to third parties exceeding normal patterns
```

### Documentation Requirements
For each suspected dissipation transaction:
- [ ] Date of transaction
- [ ] Amount
- [ ] Source account
- [ ] Destination (if transfer)
- [ ] Description from bank statement
- [ ] Comparison to historical spending patterns
- [ ] Whether during or before separation
- [ ] Opposing party's explanation (if any)
