# MSC STRATEGY — SUPREME COURT PRACTICE

## PURPOSE

Strategic guide for maximizing the probability of MSC granting leave,
accepting an original proceeding, or issuing an extraordinary writ.
MSC accepts ~3-5% of leave applications — strategy is everything.

---

## 1. GROUNDS ANALYSIS FRAMEWORK

### Step 1: Map Each Question to MCR 7.305(B) Grounds

Every question presented MUST connect to at least one ground.

```
GROUNDS MAPPING TEMPLATE:
  Question Presented: [Question]
  Primary Ground: MCR 7.305(B)([X]) — [Description]
  Secondary Ground: MCR 7.305(B)([Y]) — [Description]
  Supporting Authority: [Case/rule showing ground is met]
  Counter-Argument: [Why MSC might reject this ground]
  Strength: [STRONG / MODERATE / WEAK]
```

### Step 2: Rank Questions by Strength of Grounds

| Ranking | Criteria |
|---------|----------|
| A-Tier | Multiple strong grounds; clear conflict with published MSC decision; constitutional dimension |
| B-Tier | One strong ground; conflict with COA panel decision; significant public interest |
| C-Tier | Single ground; (B)(5) clear error only; case-specific without broader import |
| D-Tier | No clear ground; factual dispute framed as legal error; MSC likely to deny |

**Rule: Only file A-Tier and B-Tier questions. Cut C-Tier and below.**

### Step 3: Test Each Question Against MSC Denial Patterns

MSC denies leave when:
1. Issue is fact-bound with no broader legal significance
2. Record is inadequate to review the claimed error
3. Question presented is poorly framed or conclusory
4. COA decision is well-reasoned and consistent with precedent
5. Issue was not preserved in the lower courts
6. Application is untimely or procedurally defective

## 2. ISSUE SELECTION — THE DISCIPLINE OF LESS

### Why Fewer Issues Win

MSC justices review hundreds of applications per term. They decide quickly
whether an application deserves full consideration.

- **1-2 questions**: Signals confidence and focus. Forces the court to engage.
- **3 questions**: Acceptable maximum. Must be truly distinct issues.
- **4+ questions**: Signals desperation. Suggests no single issue is strong enough.

### Issue Selection Rules

```
SELECTION CRITERIA:
  1. Does this issue REQUIRE MSC resolution? (Not just "could benefit from")
  2. Does this issue affect persons/cases BEYOND this dispute?
  3. Is this issue PRESERVED in the record with clear objection?
  4. Can I state this issue in ONE sentence that a non-specialist would understand?
  5. Does this issue connect to an MCR 7.305(B)(1)-(4) ground? (Not just (B)(5))

If ANY answer is NO → consider cutting the issue.
```

### Lane-Specific Issue Selection

**Lane A (Watson/Custody):**
- STRONG: Due process in custody proceedings — constitutional dimension + broad impact
- STRONG: Parental alienation framework — unsettled Michigan law + public significance
- MODERATE: Judicial bias claim — requires strong record evidence
- WEAK: Factual custody disputes — MSC will not retry facts

**Lane B (Shady Oaks/Housing):**
- STRONG: Habitability standards — affects all Michigan tenants + developing area
- STRONG: Constructive eviction doctrine — need for uniform statewide rule
- MODERATE: Code enforcement failures — may be fact-bound
- WEAK: Damages calculation — fact-intensive, not legal question

**Lane C (Convergence/County):**
- STRONG: Judicial accountability / superintending control — constitutional dimension
- STRONG: Systemic due process violations — broad impact
- MODERATE: Governmental immunity scope — well-developed law but application matters
- WEAK: Individual rulings — case-specific without broader import

## 3. RECORD OPTIMIZATION

### What MSC Needs in the Record

MSC decides on the record below. Optimize it BEFORE filing.

```
RECORD OPTIMIZATION CHECKLIST:
  [ ] Transcripts — ALL relevant hearings ordered and filed
  [ ] Key exhibits — admitted into evidence at trial level
  [ ] Objections — EVERY claimed error preserved by timely objection
  [ ] Motions — All relevant motions and orders included
  [ ] Register of Actions — complete and certified
  [ ] COA briefs — included in the record
  [ ] COA opinion/order — certified copy obtained
```

### Building the Record for MSC Review

**At Trial Level (Prospective):**
1. Make CLEAR objections on the record — state the legal basis
2. File written motions for every substantive issue
3. Request findings of fact and conclusions of law
4. Ensure court reporter captures EVERYTHING
5. Mark and offer all critical exhibits

**At COA Level:**
1. Raise every issue you might want MSC to review
2. Ensure settled record is complete before COA briefing
3. File register of actions
4. Object to any record omissions per MCR 7.210(C)

**For MSC Application:**
1. Compile appendix from the existing record — no new evidence
2. Select excerpts that DIRECTLY support questions presented
3. Include adverse rulings you challenge
4. Include COA opinion in full

## 4. TIMING STRATEGIES

### The 56-Day Clock — MCR 7.305(C)(2)

```
TIMELINE MANAGEMENT:

Day 0:   COA decision issued
Day 1-7: Analyze COA decision — identify MSC-worthy issues
Day 7:   DECISION POINT — File motion for reconsideration in COA?
         IF YES → 56-day clock resets to ruling on reconsideration
         IF NO  → 56-day clock continues from Day 0
Day 7-21: Draft questions presented + grounds analysis
Day 21-35: Draft full application brief
Day 35-42: Compile appendix + forms
Day 42-49: Review, edit, finalize — run FILE_READY checklist
Day 49-52: Serve all parties
Day 52-54: FILE with MSC (2-day buffer for mailing)
Day 56:   ABSOLUTE DEADLINE — application must be RECEIVED
```

### Motion for Reconsideration in COA — Strategic Decision

**File reconsideration if:**
- COA overlooked a critical argument or authority
- COA made a factual error that could change the outcome
- You need more time to prepare MSC application (reconsideration resets clock)

**Do NOT file reconsideration if:**
- COA decision was well-reasoned and comprehensive
- Delay would increase harm (especially in custody cases)
- Record is already clear and complete
- Your MSC arguments don't depend on COA correcting itself

### Emergency Timing

For bypass applications and extraordinary writs — there is no fixed deadline,
but SPEED is essential:

```
EMERGENCY FILING TIMELINE:
Day 0:   Triggering event (emergency custody change, judicial overreach, etc.)
Day 1-3: Draft petition + emergency brief
Day 3-5: Compile supporting evidence
Day 5-7: FILE with MSC + serve respondent
         Request expedited consideration in cover letter
         Call clerk's office to flag emergency
```

## 5. PERSUASION AT THE MSC LEVEL

### What Moves MSC Justices to Grant Leave

1. **Institutional significance** — "This case will fix a problem affecting many cases"
2. **Clear conflict** — "Two panels said opposite things about the same law"
3. **Constitutional magnitude** — "Fundamental rights are at stake"
4. **Clarity of error** — "The lower court clearly got the law wrong"
5. **Practical consequences** — "Without MSC intervention, real harm continues"

### What Does NOT Move MSC Justices

1. Emotional appeals without legal grounding
2. Factual disputes dressed up as legal questions
3. Attacks on the lower court judge's character
4. Repetition of COA arguments without showing why MSC review is needed
5. Shotgun approach with many weak issues

### Tone and Voice for Pro Se MSC Filings

```
DO:
  - Write with precision and economy
  - Cite authority for every legal proposition
  - Acknowledge adverse authority and distinguish it
  - Frame issues in terms of statewide importance
  - Use professional, respectful language throughout
  - Show you understand the MSC's institutional role

DO NOT:
  - Use ALL CAPS for emphasis (use italics sparingly)
  - Attack judges personally (challenge rulings, not judges)
  - Include lengthy background not relevant to questions presented
  - Assume the court knows your case (they don't)
  - File multiple applications on the same issue
  - Exceed page limits (instant credibility loss)
```

## 6. PEREMPTORY REVERSAL STRATEGY

MSC can reverse WITHOUT granting full leave — a "peremptory order."

### When MSC Issues Peremptory Orders
- Error is so clear that full briefing is unnecessary
- Lower court applied clearly wrong legal standard
- Conflict with binding MSC precedent is obvious
- Issue is narrow and record is clear

### How to Position for Peremptory Reversal
1. Frame the error as a straightforward legal question (not complex/fact-bound)
2. Show the controlling authority is directly on point
3. Keep the application brief and focused
4. Show that full briefing would not change the analysis
5. Explicitly request peremptory reversal in Relief Requested section

## 7. STRATEGIC RISK ASSESSMENT

### Before Filing Any MSC Action

```
RISK MATRIX:
  Probability of Grant: [%] — based on grounds strength
  Probability of Peremptory Relief: [%] — based on error clarity
  Downside Risk: [Description] — adverse precedent, wasted resources
  Opportunity Cost: [Description] — what else could be done with this effort
  Timeline Risk: [Description] — deadline pressure, delay to other actions
  
DECISION:
  [ ] FILE — strong grounds, clear vehicle, manageable risk
  [ ] DEFER — wait for better posture (e.g., COA decision)
  [ ] ALTERNATIVE — pursue different vehicle or court
  [ ] DO NOT FILE — grounds insufficient, risk too high
```
