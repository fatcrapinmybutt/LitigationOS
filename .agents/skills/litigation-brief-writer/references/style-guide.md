# Style Guide — Michigan Court Filing Standards

Complete formatting and style reference for all Michigan court filings. Covers state circuit court, Court of Appeals, Michigan Supreme Court, and federal (W.D. Michigan) requirements.

## I. Universal Formatting Requirements

### Font and Spacing

| Element | Requirement | Authority |
|---|---|---|
| Font | Times New Roman, 12-point | MCR 7.212(B); local practice |
| Line Spacing | Double-spaced (body text) | MCR 7.212(B) |
| Single-spaced elements | Block quotes, footnotes, headings, captions | Standard practice |
| Margins | 1 inch on all sides (top, bottom, left, right) | MCR 7.212(B); local practice |
| Justification | Left-aligned (ragged right) preferred; full justification acceptable | No specific rule; left-aligned is more readable |
| Page Numbers | Centered at bottom, Arabic numerals starting at page 1 | Standard practice |
| Paper Size | 8.5" × 11" (Letter) | Universal |

### Caption Format (MCR 2.113)

Every Michigan court filing must include a caption conforming to MCR 2.113:

```
STATE OF MICHIGAN
IN THE [14TH / ___] JUDICIAL CIRCUIT COURT
FOR THE COUNTY OF [MUSKEGON / ___]

[PARTY NAME],                          Case No. [XXXX-XXXX-XX]
        Plaintiff,
                                       Hon. [Judge Name]
v.

[PARTY NAME],
        Defendant.
_________________________________/

[TITLE OF DOCUMENT]
```

**Pro Se Caption Addition:**
```
Andrew Pigors (P12345)                 [Note: Pro se litigants don't have
123 Main Street                         a P-number. Omit or write "Pro Se"
Muskegon, MI 49441                      where bar number would go.]
(231) 555-0100
andrew.pigors@email.com
Pro Se Plaintiff
```

## II. Signature Block

### Pro Se Signature Block — Michigan State Court

```
Respectfully submitted,

_______________________
Andrew Pigors, Pro Se
[Street Address]
[City, MI ZIP]
Telephone: (XXX) XXX-XXXX
Email: [email address]
Dated: [Month Day, Year]
```

### Verification (When Required)

MCR 2.114(A) requires all pleadings to be signed. Some filings require verification (sworn affirmation):

```
VERIFICATION

I, Andrew Pigors, declare under the penalties of perjury that the
facts stated in the foregoing [Document Name] are true to the best
of my information, knowledge, and belief.

_______________________
Andrew Pigors
Dated: [Month Day, Year]
```

**When Verification Is Required:**
- Complaints seeking prejudgment remedies
- Affidavits in support of motions
- Petitions (domestic relations matters may require)
- Any filing the court specifically orders to be verified

## IV. Citation Format

### Michigan Case Citations

**Standard Format:**
```
Case Name v Case Name, Volume Mich [App] Page, Pinpoint Page (Year).
```

**Examples:**
- Michigan Supreme Court: `Vodvarka v Grasmeyer, 259 Mich App 499, 508 (2003).`
- Court of Appeals: `Shade v Wright, 291 Mich App 17, 31 (2010).`
- Unpublished COA (use carefully): `Smith v Jones, unpublished per curiam opinion of the Court of Appeals, issued [Date] (Docket No. XXXXXX).`

**Subsequent References:**
- Same case, same page: `Id.`
- Same case, different page: `Id. at 512.`
- Previously cited case: `Vodvarka, 259 Mich App at 510.`

### Michigan Statute Citations

```
MCL [section number].
```
**Examples:**
- `MCL 722.27(1)(c).` (Child Custody Act)
- `MCL 554.613.` (Security Deposit Act)
- `MCL 445.911(2).` (MCPA damages)
- `MCL 600.1483.` (Non-economic damages cap — med mal)

### Michigan Court Rule Citations

```
MCR [rule number].
```
**Examples:**
- `MCR 2.113.` (Caption requirements)
- `MCR 7.212(B).` (Appellate brief format)
- `MCR 3.206(C).` (Domestic relations — motion practice)

### Federal Citations

```
Case Name, Volume U.S. Page, Pinpoint (Year).
Case Name, Volume F.3d/F.4th Page, Pinpoint (Circuit Year).
```
**Examples:**
- `Haines v Kerner, 404 U.S. 519, 520 (1972).`
- `42 U.S.C. § 1983.`
- `Fed. R. Civ. P. 12(b)(6).`

### Signal Usage

| Signal | Meaning | Use |
|---|---|---|
| [no signal] | Directly supports the proposition | Primary authority |
| *See* | Clearly supports but requires inference | Supporting authority |
| *See also* | Additional support; not primary | Cumulative authority |
| *Accord* | Different jurisdiction reaching same result | Persuasive authority |
| *Cf.* | Analogous support by different reasoning | Distinguishable but useful |
| *But see* | Directly contradicts | Counterauthority (address proactively) |
| *See generally* | Background/contextual support | Secondary sources |

## VII. Page Limits

| Court / Filing Type | Page Limit | Word Limit Alternative | Authority |
|---|---|---|---|
| COA — Appellant's Brief | 50 pages | 16,000 words | MCR 7.212(B) |
| COA — Appellee's Brief | 50 pages | 16,000 words | MCR 7.212(B) |
| COA — Reply Brief | 25 pages | 8,000 words | MCR 7.212(B) |
| COA — Application for Leave | 50 pages | 16,000 words | MCR 7.205(D) |
| MSC — Application | 50 pages | 16,000 words | MCR 7.305(A) |
| Circuit Court motions | No specific rule; 20-25 pages typical | Judge-specific | Check local practice |
| Federal (W.D. Mich) brief | 25 pages | LCivR 7.1(d)(2) | LCivR 7.1 |

## VIII. Common Formatting Errors to Avoid

1. **Wrong font size in footnotes** — Must be at least 10-point (12-point preferred)
2. **Missing page numbers** — Every page must be numbered
3. **Incorrect margins** — Many word processors default to 1.25" left/right; change to 1"
4. **Non-searchable PDFs** — MiFile prefers text-searchable PDFs; avoid image-only scans
5. **Missing caption on continuation pages** — Multi-page filings should have abbreviated caption or case number on every page header
6. **Wrong case number format** — Verify you're using the court's numbering format exactly
7. **Missing "Respectfully submitted"** — Standard closing before signature block
8. **Undated filing** — Always include the date on the signature block and Certificate of Service
