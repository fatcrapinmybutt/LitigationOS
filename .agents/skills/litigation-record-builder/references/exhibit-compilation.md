# Exhibit Compilation — Organization and Indexing Standards

Complete reference for preparing, organizing, indexing, and presenting exhibits in Michigan court proceedings, both at trial level and for appellate record. Specific to Pigors v Watson practice.

## I. Exhibit Numbering and Labeling

### Numbering Conventions

Michigan practice varies by county and judge. The two standard systems:

**System 1 — Letter-Based (Most Common in Muskegon County):**
- Plaintiff's Exhibits: A, B, C, ... Z, AA, BB, CC...
- Defendant's Exhibits: 1, 2, 3, ... (or vice versa)

**System 2 — Number-Based:**
- Plaintiff's Exhibits: 1, 2, 3...
- Defendant's Exhibits: 101, 102, 103... (or 200-series)

**Sub-Numbering for Multi-Part Exhibits:**
- Exhibit A-1, A-2, A-3 (pages or sub-documents within Exhibit A)
- Exhibit 1a, 1b, 1c (alternative sub-numbering)

### Pre-Marking Requirements

Most Michigan courts expect exhibits to be pre-marked before the hearing:

1. **Exhibit Labels**: Self-adhesive stickers placed on the first page of each exhibit
2. **Label Information**: "Plaintiff's Exhibit [Letter/Number]" + case number
3. **Placement**: Upper right corner of first page, not obscuring content
4. **Color**: Labels should match party color convention (yellow for plaintiff)

### Exhibit Label Template

```
┌─────────────────────────────────┐
│   PLAINTIFF'S EXHIBIT ___       │
│                                 │
│   Case No: ____________         │
│   Date Marked: ________         │
│   Admitted: □ Yes  □ No         │
│   Pages: ____                   │
└─────────────────────────────────┘
```

### Pre-Hearing Exhibit List Filing

Michigan practice encourages (and some judges require) filing an exhibit list before the hearing:

- **When**: 7-14 days before hearing (check local rules or judge's practice)
- **How**: File via MiFile with a brief cover letter
- **Serve**: On opposing counsel/party simultaneously
- **Purpose**: Allows opponent to review, prepare objections, and stipulate to authenticity

## IV. Binder Organization

### Physical Binder Assembly

```
EXHIBIT BINDER STRUCTURE:

Front Cover: Cover Page (yellow, with case caption)

Tab 1: EXHIBIT INDEX (full list with columns)

Tab 2: EXHIBIT A — [Description]
        ├── Exhibit label (upper right, page 1)
        ├── Document pages (numbered if multi-page)
        └── [All pages of this exhibit]

Tab 3: EXHIBIT B — [Description]
        ├── Exhibit label
        └── Document pages

[Continue for each exhibit]

Back Cover: Blank or case caption
```

### Binder Specifications

| Specification | Recommendation |
|---|---|
| Binder type | 3-ring, D-ring preferred (lies flat) |
| Binder size | 1" for small sets; 1.5"-2" for larger sets |
| Tab dividers | Lettered or numbered (matching exhibit system) |
| Tab labels | Typed labels preferred; handwritten acceptable |
| Page protectors | For original/fragile documents only (not all pages) |
| Copies needed | 1 for court, 1 for opposing party, 1 for witness, 1 for yourself = 4 minimum |

### Electronic Exhibit Preparation

For MiFile submissions or courts accepting electronic exhibits:

1. **PDF format**: Each exhibit as a separate, searchable PDF
2. **File naming**: `Plaintiff_Ex_A_Custody_Order_20250110.pdf`
3. **Bookmarks**: Add PDF bookmarks for multi-page exhibits
4. **OCR**: Run OCR on scanned documents to make them text-searchable
5. **File size**: Keep each file under 25 MB (MiFile limit)
6. **Combined file**: For some courts, combine all exhibits into one PDF with bookmarks

## V. Authentication Requirements by Exhibit Type

### MRE 901 — Authentication Framework

Every exhibit must be authenticated before admission. The proponent must produce evidence sufficient to support a finding that the item is what the proponent claims it is.

| Exhibit Type | Authentication Method | Foundation Questions |
|---|---|---|
| **Court Orders** | Self-authenticating (MRE 902(1)) | Certified copy; no testimony needed |
| **Business Records** | MRE 803(6) + MRE 901(b)(9) | Custodian testimony or certification |
| **Personal Letters** | Testimony of author or recipient (MRE 901(b)(1)) | "Do you recognize this letter? Did you write/receive it?" |
| **Photographs** | Testimony that photo fairly depicts scene (MRE 901(b)(1)) | "Does this photo accurately show [location] as it appeared on [date]?" |
| **Text Messages** | Screenshot + device identification (MRE 901(b)(1), (4)) | "Is this your phone? Do you recognize this text conversation? Is this the number for [person]?" |
| **Emails** | Printout with headers + identification (MRE 901(b)(1), (4)) | "Do you recognize this email? Did you send/receive it? Is this the email address of [person]?" |
| **Social Media** | Screenshot + profile identification + content knowledge (MRE 901(b)(1)) | "Do you recognize this profile? Is this [person's] account? When did you capture this screenshot?" |
| **Medical Records** | Business records certification (MRE 803(6)) or stipulation | Records custodian affidavit or live testimony |
| **Financial Records** | Business records (MRE 803(6)) or self-produced records + testimony | Bank certification or personal records foundation |
| **Audio/Video** | Testimony identifying content + fair representation (MRE 901(b)(1)) | "Were you present? Does this recording accurately capture what occurred?" |

### Stipulations to Authenticity

Before the hearing, contact opposing counsel/party and attempt to stipulate to the authenticity of exhibits:

**Stipulation Request Letter Template:**
```
Dear [Opposing Party/Counsel]:

Enclosed please find Plaintiff's Exhibit List and copies of all
exhibits Plaintiff intends to offer at the [hearing type] hearing
on [date].

To facilitate an efficient hearing, I request that you review these
exhibits and advise whether you will stipulate to the authenticity
(not the truth of the contents) of any of the listed exhibits.

Please respond by [date — at least 7 days before hearing].

Sincerely,
Andrew Pigors
```

## VI. Cross-Referencing Exhibits to Briefs

### In-Brief Exhibit Citations

When referencing exhibits in a brief or motion:

**Standard Format:**
```
(Ex. [Letter/Number], [Brief Description], [Date if helpful], at [Page] [if multi-page].)
```

**Examples:**
- `(Ex. A, Custody Order, at ¶ 4.)` — specific paragraph
- `(Ex. C, Text Messages, Mar. 15, 2025 entry.)` — specific entry within compilation
- `(Ex. E, Housing Photos, pp. 3-5.)` — page range within exhibit
- `(Ex. K, Building Inspection Report, at 2 [noting electrical hazard].)` — page with parenthetical

### Cross-Reference Table for Brief Drafting

Create this table before writing the brief to ensure every factual assertion is supported:

| Brief Section | Factual Assertion | Exhibit | Page/¶ | Status |
|---|---|---|---|---|
| Facts ¶ 3 | Custody order required 6 PM exchange | Ex. A | ¶ 4 | ✓ Admitted |
| Facts ¶ 7 | Defendant denied time on March 3 | Ex. D | p. 1 | ✓ Admitted |
| Arg. I.A | Pattern of 14 denials | Ex. D | pp. 1-6 | ✓ Admitted |
| Arg. II | Housing defects documented | Ex. K | pp. 1-4 | ✓ Admitted |

## VII. Appellate Exhibit Handling

### Exhibits in the Record on Appeal

All exhibits admitted or offered at trial/hearing are part of the record. However:

1. **Verify custody**: Confirm the trial court clerk has all physical exhibits
2. **Electronic exhibits**: Ensure electronic files are preserved and accessible
3. **Oversized exhibits**: Maps, diagrams, and other oversized items may need special handling
4. **Sensitive exhibits**: Sealed or confidential exhibits may require special transmission procedures

### Exhibits in the Appellate Brief Appendix

Under MCR 7.212(C), include key exhibits in the appendix:
- Only exhibits directly referenced in the brief
- Reproduce at actual size or legibly reduced
- Cross-reference to appendix page numbers in the brief body
- Do not duplicate the entire exhibit set — select the critical ones

### Exhibit Reproduction Quality

- **Legibility**: Every word must be readable in the reproduction
- **Photos**: Reproduce in color if color is relevant; black and white if not
- **Highlighted documents**: If you highlighted portions, ensure highlights are visible in copies
- **Small text**: Consider enlarging critical portions in a demonstrative appendix page
